## 概述

本文介绍了一个面向通用人形机器人全身移动操作任务的开源基础模型 Psi0（Ψ₀）。该研究的主要动机在于解决当前人形机器人系统中高层语义规划与低层运动执行之间存在显著断点的问题，通过构建一个统一的视觉-语言-动作（VLA）模型框架，实现从语言指令到关节级动作序列的端到端映射。Psi0 的核心贡献在于提出了一种将异构动捕数据与合成数据、视觉语言模型（VLM）语义规划/路由、以及潜变量/动作 token 预测机制整合在同一条训练与部署链路中的方法，从而为通用人形操作能力提供了一个开放、可扩展的基础模型范式。

## 核心内容

（a）研究背景与问题：人形机器人因其类人形态而具备在人类环境中执行复杂操作任务的巨大潜力，但实现真正的“通用”移动操作仍面临严峻挑战。现有方法通常将高层任务规划（如自然语言理解、场景语义解析）与低层运动控制（如关节角度、力矩指令）割裂为独立模块，导致信息传递中的语义丢失与执行断点。此外，人形机器人的高自由度、全身协调需求以及操作与移动的耦合特性，使得传统基于单一数据源或手工规则的方法难以泛化到多样化的真实场景。因此，如何构建一个能够无缝连接高层认知与低层动作的端到端模型，成为该领域的关键瓶颈。

（b）方法或模型框架：Psi0 提出了一种多模态编码-预测架构。首先，系统将语言指令、相机图像/多视角观测、机器人本体状态（如关节角度、IMU 数据）以及历史关节序列统一编码为多模态表征。随后，该表征被送入一个联合训练链路，该链路包含两个核心组件：一是基于视觉语言模型（VLM）的语义规划/路由模块，负责将高层任务目标分解为可执行的子目标或操作原语；二是基于潜变量与动作 token 的预测模块，负责生成具体的动作块（action chunk）或动作 token 序列。通过将异构动捕数据（如人类演示）与合成数据（如仿真生成）同时纳入训练，模型能够学习到丰富的运动先验与任务相关性。

（c）关键技术创新：Psi0 的核心创新在于其“异构动捕与合成平衡数据”与“VLM 语义规划/路由”被置于同一条训练/部署链路中，而非作为独立的前后处理步骤。这种设计使得高层语义信息能够直接指导低层动作 token 的生成，同时低层运动反馈也能反向影响语义规划，从而显著减少了从目标到动作之间的断点。此外，模型采用潜变量机制来捕捉动作分布中的多模态性（例如，同一任务有多种可行的操作轨迹），并通过动作 token 的离散化预测实现了高效的序列生成，兼顾了表达的丰富性与计算的可行性。

（d）实验/验证或应用价值：虽然论文元数据未提供具体的定量实验结果，但从其设计范式可以推断，Psi0 旨在为通用人形操作提供一个可复现、可扩展的基础模型。其开源性质（项目页面已公开）意味着研究社区可以在其基础上进行微调、数据增强或任务扩展。潜在的应用场景包括家庭服务（如整理物品、开门）、工业装配（如零件抓取与放置）以及危险环境中的远程操作。通过将 VLM 的语义理解能力与运动生成能力深度融合，Psi0 有望推动人形机器人从特定任务的“编程执行”向“语言指令驱动的自主操作”迈进，为构建真正的通用机器人智能体奠定模型基础。
