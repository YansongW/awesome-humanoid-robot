TextOp: Real-time Interactive Text-Driven Humanoid Robot Motion Generation and Control

TextOp: Real-time Interactive Text-Driven Humanoid Robot Motion Generation and Control

지능 계층
검증
초안

요약

TextOp 主要解决数据闭环：用语言指令、遥操作/外骨骼数据采集人类操作和机器人状态，再通过扩散策略/流匹配、全身控制器/WBC/MPC转成可训练、可复用的全身轨迹/动作序列。关键点是把动作生成看成条件生成问题，用扩散或流匹配在多模态动作分布里采样可执行轨迹。

출처

  • TextOp project page

개요

TextOp는 인간형 로봇의 동작 생성에서 발생하는 데이터 폐쇄 루프 문제를 해결하기 위해 설계되었습니다. 언어 명령, 원격 조작, 외골격을 통해 수집된 인간 조작 데이터 및 로봇 상태 정보를 통합하고, 확산 전략 또는 흐름 매칭 방법을 전신 제어기(WBC/MPC)와 결합하여 다중 모달 데이터를 훈련 가능하고 재사용 가능한 전신 궤적 및 동작 시퀀스로 변환합니다. 이 연구의 핵심 기여는 동작 생성을 조건부 생성 문제로 모델링하여 다중 모달 동작 분포에서 실행 가능한 궤적을 효율적으로 샘플링하고, 실시간 대화형 텍스트 기반 동작 제어를 가능하게 하는 데 있습니다.

핵심 내용

현재 인간형 로봇의 동작 생성은 두 가지 주요 과제에 직면해 있습니다. 첫째는 인간 시연 데이터와 로봇 상태 정보를 효과적으로 활용하여 폐쇄 루프 데이터 흐름을 구축하는 방법이고, 둘째는 자연어 명령을 실행 가능한 전신 동작으로 변환하는 방법입니다. 전통적인 방법은 수동으로 설계된 동작 라이브러리나 사전 프로그래밍된 궤적에 의존하는 경우가 많아 복잡하고 변화무쌍한 상호작용 시나리오에 적응하기 어렵습니다. TextOp는 이 문제를 해결하기 위해 동작 생성을 조건부 생성 문제로 간주하고, 확산 모델 또는 흐름 매칭 기술을 활용하여 다중 모달 동작 분포에서 언어 명령에 부합하는 전신 궤적을 샘플링함으로써 언어 이해에서 동작 실행까지의 완전한 연결 고리를 구축합니다.

방법론적 프레임워크 측면에서 TextOp는 종단 간 실시간 동작 생성 및 제어 시스템을 구축합니다. 시스템은 먼저 원격 조작 또는 외골격 장치를 통해 인간 조작 데이터를 수집하고 동시에 로봇 상태 정보를 기록합니다. 이 데이터는 통일된 궤적 표현으로 변환되어 확산 전략 또는 흐름 매칭 모델의 훈련 데이터로 입력됩니다. 모델은 언어 명령을 조건으로 동작 분포에서 전신 관절 궤적을 샘플링한 후, 전신 제어기(WBC) 또는 모델 예측 제어(MPC)가 하위 수준 추적 및 안정화 제어를 수행합니다. 이 프레임워크는 데이터 수집, 모델 훈련에서 실시간 제어에 이르는 데이터 폐쇄 루프를 구현합니다.

핵심 기술 혁신은 세 가지 수준에서 나타납니다. 첫째, 확산 전략과 흐름 매칭을 인간형 로봇 동작 생성에 도입하여 모델이 다중 모달, 고차원 동작 분포를 처리하고 자연스럽고 다양한 동작을 생성할 수 있도록 합니다. 둘째, 언어 명령을 조건 입력으로 사용하여 제로샷 또는 소수 샘플에서 동작 일반화를 가능하게 하여 사용자가 프로그래밍 없이 자연어를 통해 로봇이 복잡한 동작을 수행하도록 제어할 수 있습니다. 셋째, 전신 제어기와 모델 예측 제어를 결합하여 생성된 궤적의 물리적 실현 가능성과 실시간성을 보장하고 시스템이 동적 환경에서 안정적으로 작동할 수 있도록 합니다.

실험 검증 측면에서 TextOp는 보행, 잡기, 자세 조정 등 전형적인 작업을 포함한 다양한 인간형 로봇 플랫폼에서 테스트되었습니다. 실험 결과는 시스템이 언어 명령에 따라 실시간으로 일관된 전신 동작을 생성할 수 있으며, 원격 조작 데이터가 제한된 경우에도 높은 생성 품질을 유지함을 보여줍니다. 또한 순수 운동학적 방법과 비교하여 TextOp는 궤적 평활도, 작업 완료율 및 인간-로봇 상호작용의 자연스러움에서 현저한 우위를 보입니다. 이 연구는 미래의 인간-로봇 협업, 가사 서비스 등 시나리오에서 인간형 로봇 응용을 위한 실현 가능한 기술 경로를 제공합니다.

응용 가치 측면에서 TextOp는 인간형 로봇 동작 프로그래밍의 진입 장벽을 낮추어 비전문 사용자도 자연어를 통해 로봇과 상호작용할 수 있도록 합니다. 동시에 데이터 폐쇄 루프 설계는 지속적 학습과 모델 반복을 위한 기반을 마련하여 인간형 로봇이 실험실에서 실제 응용으로 나아가는 것을 촉진할 것으로 기대됩니다. 이 연구는 인공지능과 로봇 공학의 교차 분야에서 중요한 시범적 의미를 가지며, 특히 언어 기반 전신 동작 생성 및 제어를 위한 체계적인 솔루션을 제공합니다.
