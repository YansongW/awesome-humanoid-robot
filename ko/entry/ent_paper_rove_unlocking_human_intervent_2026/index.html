ROVE: Unlocking Human Interventions for Humanoid Manipulation via Reinforcement Learning

ROVE: Unlocking Human Interventions for Humanoid Manipulation via Reinforcement Learning

영역
AI 모델 및 알고리즘소프트웨어 미들웨어
계층
지능 계층

초안

요약

ROVE 的实现路径是先把语言指令、相机图像/多视角观测、本体状态与关节序列编码成多模态表征，再用PPO/RL 策略训练、扩散策略/流匹配、VLA 多模态动作模型预测可执行动作命令。关键点是保留 VLM 的语义理解，同时增加机器人状态和动作头，避免只停留在语言规划。

  • ROVE project page

개요

휴머노이드 로봇이 복잡한 환경에서 정밀한 조작 작업을 수행할 때, 고수준의 의미 이해와 저수준의 운동 제어를 효과적으로 결합하는 방법이 핵심 과제로 대두되고 있다. 기존 방법들은 시각-언어 모델(VLM)의 언어 계획과 동작 실행 사이에 괴리가 존재하여, 인식에서 행동으로 이어지는 종단 간 폐쇄 루프를 구현하기 어렵다. ROVE는 강화 학습 기반 프레임워크를 제안하며, 언어 명령, 다중 시점 시각 관측, 본체 상태 및 관절 시퀀스를 통합된 다중 모달 표현으로 인코딩하고, PPO 강화 학습 전략, 확산 전략 및 VLA 다중 모델 동작 모델을 융합하여 인간의 개입 신호를 효과적으로 활용한다. 이 연구의 핵심 기여는 VLM의 의미 이해 능력을 유지하면서 로봇 상태와 동작 헤드를 도입하여, 기존 방법이 언어 계획 수준에 머무르는 한계를 돌파한 점에 있다.

핵심 내용

연구 배경 및 문제점: 휴머노이드 로봇의 범용 조작 능력은 자율 에이전트로 나아가는 핵심 요소이지만, 기존 방법들은 복잡하고 비구조화된 작업을 처리할 때 현저한 병목 현상에 직면한다. 한편, 시각-언어 모델 기반 방법은 의미 이해와 작업 계획에 뛰어나지만, 생성된 계획은 종종 로봇 본체의 동역학 및 실시간 환경 피드백을 고려하지 않아 정밀한 관절 수준의 동작 시퀀스로 직접 매핑하기 어렵다. 다른 한편, 순수 강화 학습 방법은 시행착오를 통해 강건한 제어 전략을 학습할 수 있지만, 고차원 상태-동작 공간에서 훈련 효율이 낮고 인간의 사전 지식을 통합하기 어렵다. VLM의 의미 추론 장점과 강화 학습의 동작 최적화 능력을 유기적으로 결합하면서, 훈련 및 실행 과정에서 인간이 효과적으로 개입할 수 있도록 하는 것이 이 분야의 핵심 과제로 남아 있다.

방법 또는 모델 프레임워크: ROVE는 다중 모달 표현 학습과 전략 최적화를 결합한 통합 프레임워크를 구축한다. 입력 계층은 언어 명령, 다중 시점 카메라 이미지, 로봇 본체 상태(관절 각도, 말단 효과기 자세 등) 및 과거 관절 시퀀스를 인코딩하여 통합된 다중 모달 임베딩 공간을 형성한다. 전략 학습 단계에서는 프레임워크가 두 가지 기술 경로를 병렬로 사용한다. 첫째, 근접 정책 최적화(PPO) 기반 강화 학습 훈련을 통해 환경과 상호작용하며 동작 전략을 지속적으로 최적화한다. 둘째, 확산 전략 또는 흐름 매칭 방법을 활용하여 부드럽고 연속적인 동작 궤적을 생성한다. 최종적으로 VLA(시각-언어-동작) 다중 모달 동작 모델이 위 두 전략의 출력을 융합하여 직접 실행 가능한 동작 명령을 예측한다. 이 프레임워크의 핵심은 공유된 의미-운동 표현 계층을 설계하여 VLM의 의미 특징이 로봇 상태 특징과 동작 공간에서 협력적으로 작용할 수 있도록 하는 데 있다.

핵심 기술 혁신: ROVE의 핵심 혁신은 "의미 이해를 유지하고 상태-동작 헤드를 추가"하는 설계 철학에 있다. 기존 VLA 방법은 종종 VLM의 출력을 직접 동작 명령으로 사용하여, 모델이 언어 수준의 계획만 수행할 뿐 물리적 세계의 동적 변화에 적응하지 못하는 한계를 보였다. ROVE는 VLM에 전용 로봇 상태 인코더와 동작 디코딩 헤드를 추가함으로써, 모델이 기존의 의미 이해 능력을 유지하면서 본체 상태를 인식하고 운동학 및 동역학 제약 조건을 충족하는 동작을 생성할 수 있도록 한다. 또한, 프레임워크에 도입된 인간 개입 메커니즘은 작업자가 훈련 과정에서 시범 또는 교정 신호를 통해 전략 탐색을 유도하여, 수렴을 가속화하고 전략의 안전성을 향상시킨다. 이러한 설계는 ROVE가 언어 일반화 능력을 희생하지 않으면서 "이해"에서 "실행"으로의 원활한 연결을 구현할 수 있게 한다.

실험/검증 및 응용 가치: 논문이 구체적인 실험 데이터를 제공하지는 않았지만, 프레임워크 설계로 미루어 보아 ROVE는 정밀 조작과 의미 이해가 결합된 시나리오, 예를 들어 물체 잡기, 도구 사용, 조립 작업 등에 적합할 것으로 추정된다. 다중 시점 관측 하에서 강화 학습과 확산 전략을 융합함으로써, 이 방법은 시뮬레이션 환경과 실제 로봇 플랫폼에서 순수 VLM 계획 또는 순수 RL 방법보다 더 높은 작업 성공률과 샘플 효율을 보일 것으로 기대된다. 그 응용 가치는 휴머노이드 로봇에 확장 가능한 범용 조작 패러다임을 제공하는 데 있다. 한편으로 VLM의 의미 이해 능력은 로봇이 자연어 명령을 이해하고 새로운 작업에 적응할 수 있게 하며, 다른 한편으로 강화 학습과 인간 개입의 결합은 동작의 물리적 실현 가능성과 강건성을 보장한다. 이 프레임워크는 향후 휴머노이드 로봇이 가사 서비스, 산업 협력 등 분야에 배치되는 데 중요한 기술적 경로를 제시한다.
