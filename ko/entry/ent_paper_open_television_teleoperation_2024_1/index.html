요약

Open-TeleVision 主要解决数据闭环：用相机图像/多视角观测、本体状态与关节序列、遥操作/外骨骼数据采集人类操作和机器人状态，再通过ACT/行为克隆模仿学习、扩散策略/流匹配、MM-DiT/Transformer 动作头转成可训练、可复用的关节位置/力矩命令、末端执行器/腕手目标、动作 chunk/token。关键点是把动作生成看成条件生成问题，用扩散或流匹配在多模态动作分布里采样可执行轨迹。

개요

Open-TeleVision은 휴머노이드 로봇을 위한 몰입형 능동 시각 원격 조작 시스템을 제안하며, 로봇 데이터 수집 및 스킬 학습 과정에서의 데이터 폐쇄 루프 문제를 해결하는 것을 목표로 합니다. 이 시스템은 다중 시점 카메라 관측, 본체 상태 및 관절 시퀀스를 융합하여 인간의 조작 행동을 훈련 가능하고 재사용 가능한 로봇 동작 명령으로 변환합니다. 핵심 기여는 동작 생성을 조건부 생성 문제로 모델링하고, 확산 모델 또는 흐름 매칭 방법을 사용하여 다중 모드 동작 분포에서 실행 가능한 궤적을 샘플링함으로써 원격 조작의 데이터 품질과 모방 학습의 일반화 능력을 향상시키는 데 있습니다.

핵심 내용

연구 배경 및 문제 휴머노이드 로봇의 정밀 조작 및 자율 행동 학습은 고품질의 대규모 시연 데이터에 크게 의존합니다. 그러나 기존 원격 조작 방식은 시각적 피드백의 몰입감과 실시간성에 한계가 있어, 조작자가 로봇의 말단 실행기와 손목-손 목표를 정밀하게 제어하기 어렵습니다. 또한 로봇 동작 분포는 다중 모드 특성을 가지며, 단일 결정론적 매핑으로는 복잡한 작업에서의 다양한 실행 가능 전략을 포괄하기 어렵습니다. 기존 데이터 수집 시스템은 종종 데이터 폐쇄 루프가 단절되어, 인간 조작과 로봇 상태 간의 매핑에 재사용 가능한 생성 프레임워크가 부족하여 모방 학습(예: 행동 복제, ACT)의 일반화 성능을 제한합니다.

방법 또는 모델 프레임워크 Open-TeleVision은 종단 간 원격 조작 데이터 수집 및 동작 생성 프레임워크를 구축합니다. 시스템 전단은 다중 시점 카메라 어레이와 본체 센서를 사용하여 조작자의 동작과 로봇 상태를 실시간으로 캡처하고, 이미지 관측, 관절 시퀀스, 말단 실행기 자세를 포함하는 다중 모드 데이터 스트림을 형성합니다. 후단은 확산 정책 또는 흐름 매칭 기반의 조건부 생성 모델을 도입하여 동작 생성을 다중 모드 분포에서 실행 가능한 궤적을 샘플링하는 과정으로 간주합니다. 구체적으로, 모델은 MM-DiT(다중 모드 확산 트랜스포머) 또는 트랜스포머 동작 헤드를 통해 관측 조건을 관절 위치/토크 명령 또는 동작 조각(액션 청크/토큰)으로 매핑하여 인간 시연에서 로봇 명령으로의 훈련 가능한 변환을 실현합니다.

핵심 기술 혁신 이 연구의 혁신점은 세 가지 측면에서 나타납니다. 첫째, '몰입형 능동 시각 피드백' 메커니즘을 제안하여 조작자가 헤드 마운트 디스플레이를 통해 로봇 시점과 정렬된 1인칭 시각을 얻고 능동적 시점 전환을 지원함으로써 원격 조작의 현장감과 정밀 제어 능력을 크게 향상시킵니다. 둘째, 동작 생성 문제를 조건부 생성 모델링으로 재정의하고 확산 또는 흐름 매칭 방법을 사용하여 다중 모드 동작 분포를 처리함으로써 복잡한 작업에서 기존 결정론적 정책의 모드 붕괴 문제를 방지합니다. 셋째, 시스템 설계는 데이터 폐쇄 루프의 완전성을 강조하여 수집, 주석, 훈련, 배포까지 통합 파이프라인을 형성함으로써 수집된 원격 조작 데이터를 하류 모방 학습 정책의 사전 훈련 및 미세 조정에 직접 사용할 수 있도록 합니다.

실험/검증 및 응용 가치 논문이 구체적인 실험 데이터를 제공하지는 않지만, 시스템 아키텍처와 도메인 태그(예: deployment, real_world, sensor_suite)로부터 이 연구가 실제 로봇 플랫폼에서 배포 검증되었음을 추론할 수 있습니다. Open-TeleVision의 가치는 로봇 커뮤니티에 오픈 소스이며 재현 가능한 원격 조작 인프라를 제공하는 데 있으며, 특히 높은 정밀도의 손목-손 조작이 필요한 작업(예: 조립, 파지)에 적합합니다. 데이터 수집과 생성적 동작 모델링을 결합함으로써 이 방법은 휴머노이드 로봇 스킬 학습의 시연 비용을 낮추고 시뮬레이션에서 실제 세계로의 전환을 가속화하며, 후속 연구에 표준화된 데이터 폐쇄 루프 기준을 제공할 것으로 기대됩니다.
