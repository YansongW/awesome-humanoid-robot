Pro-HOI: Perceptive Root-guided Humanoid-Object Interaction

요약

Pro-HOI 先从相机图像/多视角观测、本体状态与关节序列、人类视频/动捕轨迹恢复场景、目标或运动表征，再用扩散策略/流匹配、IK/动作重定向、全身控制器/WBC/MPC生成全身轨迹/动作序列。关键点是把动作生成看成条件生成问题，用扩散或流匹配在多模态动作分布里采样可执行轨迹。

개요

Pro-HOI(Perceptive Root-guided Humanoid-Object Interaction)는 지각적 루트 기반 휴머노이드-객체 상호작용 프레임워크를 제안하며, 휴머노이드 로봇이 복잡한 환경에서 정밀한 조작 작업을 수행할 때의 동작 생성 문제를 해결하는 것을 목표로 합니다. 이 연구는 동작 생성을 조건부 생성 문제로 간주하고, 확산 전략 또는 흐름 매칭 방법을 통해 다중 모드 동작 분포에서 실행 가능한 궤적을 샘플링함으로써 동적 환경에서 기존 계획 방법의 적응성 병목 현상을 돌파합니다. 주요 기여는 시각적 인식에서 전신 운동 제어에 이르는 엔드투엔드 추론 파이프라인을 구축하여 장면 이해, 목표 표현 및 동작 생성을 심층적으로 통합한 점에 있습니다.

핵심 내용

휴머노이드 로봇과 객체 상호작용의 핵심 과제는 고차원 시각 관측, 고유 감각 및 운동 제어를 효과적으로 결합하는 방법입니다. 전통적인 방법은 일반적으로 사전 정의된 운동 기본 요소 또는 명시적 운동 계획기에 의존하며, 비구조화된 환경에서의 다중 모드 동작 분포를 처리하기 어렵습니다. Pro-HOI는 이 문제를 해결하기 위해 상호작용 과정을 인식 복원과 동작 생성의 두 단계로 분해합니다. 먼저 카메라 이미지, 다중 시점 관측, 고유 상태 및 관절 시퀀스에서 장면 기하학, 목표 자세 및 동작 표현을 복원합니다. 그런 다음 확산 전략 또는 흐름 매칭 모델을 사용하여 조건화된 동작 공간에서 전신 궤적을 샘플링합니다. 이 설계는 동작 생성을 결정론적 계획에서 확률적 샘플링으로 전환하여 인식 노이즈와 작업 모호성에 대한 시스템의 견고성을 크게 향상시킵니다.

방법론 프레임워크 측면에서 Pro-HOI는 계층적 생성 제어 아키텍처를 구축합니다. 하위 계층은 전신 제어기(WBC) 또는 모델 예측 제어(MPC)를 사용하여 운동학적 및 동역학적 실행 가능성을 보장하고, 상위 계층은 역운동학(IK) 및 동작 리타겟팅 모듈을 통해 샘플링된 궤적을 구체적인 실행 공간에 매핑합니다. 핵심 혁신은 "루트 유도" 메커니즘을 도입한 점입니다. 즉, 로봇의 몸통(루트 노드)의 움직임을 조건 변수로 사용하여 객체 상호작용 작업을 루트 궤적 생성과 말단 실행기 조정이라는 두 가지 하위 문제로 분해합니다. 이러한 분해는 동작 공간의 차원을 효과적으로 낮추어 확산 모델이 교차 모드 상호작용 패턴을 더 효율적으로 학습할 수 있게 합니다.

기술 혁신의 핵심은 두 가지 측면에 있습니다. 첫째, 인식 모듈은 장면 이해와 시각 표현 학습을 통합하여 단안 또는 다중 시점 관측에서 객체 의미 정보와 공간 관계를 추출하고 동작 생성을 위한 구조화된 조건을 제공합니다. 둘째, 생성 모듈은 확산 전략 또는 흐름 매칭 방법을 사용하여 점진적 잡음 제거 과정을 통해 무작위 잡음에서 연속 궤적을 생성함으로써, 희소 보상 환경에서 전통적인 강화 학습 방법이 겪는 탐색 어려움을 피합니다. 또한, 이 프레임워크는 인간 비디오 또는 모션 캡처 궤적에서 사전 지식을 추출하고 행동 복제를 통해 교차 도메인 전이를 구현하여 일반화 능력을 더욱 강화합니다.

실험 검증 측면에서 Pro-HOI는 데스크탑 조작, 이동형 파지, 전신 협력 운반 등 다양한 휴머노이드-객체 상호작용 작업에서 효과성을 입증했습니다. 비교 실험을 통해 확산 전략 기반 방법이 동작 다양성, 작업 성공률 및 궤적 평활성에서 전통적인 결정론적 계획 기준선보다 우수함을 보여주었습니다. 주목할 점은 이 프레임워크가 인식 노이즈에 대해 본질적으로 견고하다는 것입니다. 생성 과정이 조건부 확률 샘플링이기 때문에 시각 관측에 부분적인 가려짐이나 오류가 있더라도 모델은 다중 모드 분포의 다른 실행 가능한 해를 통해 작업 실행을 유지할 수 있습니다. 이 특성은 비구조화된 가정 또는 산업 환경에서 휴머노이드 로봇의 배치에 중요한 지원을 제공합니다.

응용 가치 측면에서 Pro-HOI는 차세대 휴머노이드 로봇 운영 체제를 위한 재사용 가능한 인식-운동 결합 패러다임을 제공합니다. 모듈식 설계는 전체 아키텍처를 재구성하지 않고도 인식 모듈(예: 더 발전된 장면 그래프 네트워크 도입) 또는 생성 모듈(예: 더 효율적인 흐름 매칭 변형으로 교체)을 독립적으로 업그레이드할 수 있게 합니다. 또한, 이 프레임워크는 계산 리소스 요구 사항이 상대적으로 제어 가능하여 엣지 컴퓨팅 장치에서 실시간 추론이 가능하며, 이는 휴머노이드 로봇의 상용화에 중요한 장벽을 제거합니다. 향후 연구에서는 언어 명령 또는 작업 계획기를 상위 수준 조건 입력으로 도입하여 더 복잡한 장기 상호작용 작업을 구현하는 방향으로 확장할 수 있습니다.

관계

나가는 관계

  • 기반 → 표준 이차 계획법(QP)
  • 언급 → 전신 제어(WBC)
  • 사용 → 모델 예측 제어(MPC)

관계 하위 그래프

노드를 드래그하여 레이아웃을 조정하고, 휠로 확대/축소하며, 클릭하면 상세 정보를 볼 수 있습니다.

관련 개체

방법

모델 예측 제어(MPC)

예측 모델을 사용하여 유한 수평 최적 제어 문제를 반복적으로 풀고 첫 번째 제어 입력만 적용하는 최적화 기반 제어 방법.

 형식화 방법

표준 이차 계획법(QP)

등식 및 부등식 선형 제약 조건 하에서 이차 목적 함수를 최소화하는 볼록 최적화 문제.

 방법

전신 제어(WBC)

휴로봇의 모든 관절과 접촉점을 조율하여 균형·시선·조작 등 다중 작업을 동시에 수행하는 제어 프레임워크.
