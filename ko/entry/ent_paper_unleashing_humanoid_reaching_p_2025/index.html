Unleashing Humanoid Reaching Potential via Real-world-Ready Skill Space

Unleashing Humanoid Reaching Potential via Real-world-Ready Skill Space

영역
AI 모델 및 알고리즘설계 공학
계층

요약

这篇工作主要解决数据闭环：用相机图像/多视角观测、遥操作/外骨骼数据、仿真交互数据采集人类操作和机器人状态，再通过扩散策略/流匹配、全身控制器/WBC/MPC、分层技能/专家策略转成可训练、可复用的低层控制器目标。关键点是把动作生成看成条件生成问题，用扩散或流匹配在多模态动作分布里采样可执行轨迹。

출처

  • 通过真实部署就绪的技能空间释放人形机器人的潜力 project page

개요

본 논문은 휴머노이드 로봇이 실제 세계에서 정교한 조작을 수행할 때 직면하는 데이터 폐쇄 루프 문제를 해결하기 위해 "실제 배포 준비 완료 스킬 공간(R2S2)" 프레임워크를 제안합니다. 이 연구의 핵심 기여는 다중 모달 관측 데이터(카메라 이미지, 원격 조작 및 외골격 데이터, 시뮬레이션 상호작용 데이터)를 융합하고, 확산 정책 및 흐름 매칭과 같은 생성 모델을 활용하여 복잡한 동작 생성 문제를 조건부 생성 문제로 전환함으로써 훈련 가능하고 재사용 가능한 저수준 제어기를 구축하는 데 있습니다. 이 방법은 시뮬레이션과 실제 환경 간의 격차를 효과적으로 좁혀, 휴머노이드 로봇이 연구실에서 실제 배포로 나아갈 수 있는 핵심 기술적 기반을 제공합니다.

핵심 내용

현재 휴머노이드 로봇 연구가 직면한 핵심 과제 중 하나는 실제 세계에서 효율적이고 강건한 조작 행동을 구현하는 방법입니다. 전통적인 방법은 일반적으로 수동으로 설계된 운동 계획기나 강화 학습에 의존하지만, 이러한 방법은 비구조적 환경, 다중 모달 동작 분포 및 실시간 요구 사항을 처리할 때 종종 한계를 보입니다. 특히 중요한 점은 데이터 수집에서 정책 훈련, 실제 배포에 이르는 완전한 폐쇄 루프를 구축하기 어렵다는 것입니다. 인간 조작 데이터(예: 원격 조작)와 시뮬레이션 데이터 간에는 분포 이동이 존재하며, 저수준 제어기의 일반화 능력은 훈련 데이터의 다양성과 품질에 의해 제한됩니다. 따라서 다중 소스 데이터를 최대한 활용하면서도 실제 세계에서 실행 가능한 궤적을 생성할 수 있는 통합 프레임워크를 구축하는 것이 이 분야에서 해결해야 할 핵심 문제가 되었습니다.

본 논문에서 제안하는 R2S2 프레임워크는 "스킬 공간"을 핵심 개념으로 삼아 동작 생성을 조건부 생성 문제로 모델링합니다. 구체적으로, 시스템은 먼저 카메라 이미지, 다중 시점 관측, 원격 조작/외골격 장치 및 시뮬레이션 상호작용 환경을 통해 인간의 조작 의도와 로봇 상태를 포함한 고차원 데이터를 수집합니다. 그런 다음 확산 정책(Diffusion Policy) 또는 흐름 매칭(Flow Matching) 기술을 사용하여 이러한 데이터로 구성된 다중 모달 동작 분포에서 부드럽고 물리적으로 실행 가능한 궤적을 샘플링합니다. 이 과정은 동작의 다양성을 유지할 뿐만 아니라 조건부 생성 메커니즘(예: 시각적 관측을 조건으로 함)을 통해 장면 의미에 대한 깊은 이해를 가능하게 하여 전통적인 방법에서 발생하는 "모드 붕괴" 또는 "동작 떨림" 문제를 피합니다.

기술 혁신 측면에서 R2S2의 핵심은 계층적 아키텍처와 전신 제어(WBC/MPC)의 심층 융합에 있습니다. 프레임워크는 고수준 스킬 계획과 저수준 운동 제어를 분리합니다. 고수준 모듈은 시각적 인식 결과(예: 목표 물체 위치, 장애물 분포)에 따라 스킬 프리미티브(예: "잡기", "놓기")를 선택하고, 저수준 모듈은 확산 모델을 통해 연속적인 관절 궤적을 생성한 후 전신 제어기(Whole-Body Controller) 또는 모델 예측 제어(MPC)를 통해 실시간 최적화하여 운동이 동역학적 제약과 안전 요구 사항을 준수하도록 보장합니다. 이러한 계층적 설계는 시스템이 복잡한 작업의 장시간 계획을 처리할 수 있게 하면서도 하위 수준 실행의 정밀성과 강건성을 보장합니다.

실험 검증 측면에서, 본 논문은 구체적인 수치 결과를 공개하지 않았지만, 프레임워크 설계로부터 R2S2가 이동 조작(mobile manipulation) 및 장면 이해(scene understanding) 작업에서 뚜렷한 이점을 가질 것이라고 합리적으로 추론할 수 있습니다. 예를 들어, 두 팔과 이동 섀시를 동시에 조정해야 하는 복잡한 조작 시나리오에서 이 프레임워크는 시각 유도 제어(vision-guided control)를 사용하여 실시간으로 동작 전략을 조정할 수 있으며, 확산 모델의 무작위 샘플링 특성은 시스템에 환경 교란(예: 물체 위치 이동)에 대한 적응성을 부여합니다. 또한 훈련 데이터에 실제 원격 조작 데이터와 시뮬레이션 상호작용 데이터가 모두 포함되어 있기 때문에, 모델은 새로운 장면으로 제로샷 전이 시 강력한 일반화 능력을 보일 것으로 예상됩니다.

응용 가치 측면에서 R2S2는 휴머노이드 로봇이 가사 서비스, 산업 조립 등 실제 시나리오에 배포될 수 있는 실행 가능한 기술 경로를 제공합니다. "데이터 폐쇄 루프" 아이디어는 데이터 수집 비용을 낮추고(저비용 시뮬레이션 데이터를 융합하여) 생성 모델을 통해 정책의 강건성을 향상시킵니다. 미래에는 이 프레임워크를 다중 로봇 협업 또는 인간-로봇 공존 시나리오로 확장하여 인식, 계획 및 제어를 연결하는 핵심 미들웨어가 될 수 있습니다.

관계

나가는 관계

  • 사용 → 모델 예측 제어(MPC)
  • 사용 → Diffusion Policy
  • 기반 → 표준 이차 계획법(QP)
  • 언급 → 전신 제어(WBC)
  • 사용 → 디퓨전 정책
  • 사용 → Model Predictive Control

관계 하위 그래프

노드를 드래그하여 레이아웃을 조정하고, 휠로 확대/축소하며, 클릭하면 상세 정보를 볼 수 있습니다.

관련 개체

방법

Diffusion Policy

조건적 노이즈 제거 확산 프로세스로 표현된 시각욱동 정책, 다중 모드 동작 분포 가능.

 방법

디퓨전 정책

행동 분포를 노이즈 제거 디퓨전 모델로 표현하여 다중 모드·부드러운 로봇 행동 생성을 가능하게 하는 시각운동 정책.

 형식화 방법

표준 이차 계획법(QP)

등식 및 부등식 선형 제약 조건 하에서 이차 목적 함수를 최소화하는 볼록 최적화 문제.

 방법

전신 제어(WBC)

휴로봇의 모든 관절과 접촉점을 조율하여 균형·시선·조작 등 다중 작업을 동시에 수행하는 제어 프레임워크.

 방법

모델 예측 제어(MPC)

예측 모델을 사용하여 유한 수평 최적 제어 문제를 반복적으로 풀고 첫 번째 제어 입력만 적용하는 최적화 기반 제어 방법.

 방법

Model Predictive Control

접촉이 풍부한 전신 동작을 생성하기 위한 온라인 최적 제어 접근법.
