TextOp: Real-time Interactive Text-Driven Humanoid Robot Motion Generation and Control

요약

TextOp 主要解决数据闭环：用语言指令、遥操作/外骨骼数据采集人类操作和机器人状态，再通过扩散策略/流匹配、全身控制器/WBC/MPC转成可训练、可复用的全身轨迹/动作序列、低层控制器目标。关键点是把动作生成看成条件生成问题，用扩散或流匹配在多模态动作分布里采样可执行轨迹。

개요

TextOp는 실시간 대화형 텍스트 기반 휴머노이드 로봇 동작 생성 및 제어 프레임워크를 제안하며, 로봇 동작 데이터 획득 및 재사용 과정에서의 '데이터 폐쇄 루프' 문제를 해결하는 것을 목표로 합니다. 이 연구는 동작 생성을 조건부 생성 문제로 모델링하고, 확산 전략 또는 흐름 매칭 방법을 활용하여 다중 모드 동작 분포에서 실행 가능한 궤적을 샘플링함으로써 언어 명령 및 원격 조작 데이터를 훈련 가능하고 재사용 가능한 전신 동작 시퀀스로 변환합니다. 주요 기여는 인간 조작에서 로봇 자율 동작으로 이어지는 데이터 폐쇄 루프를 구축하여 행동 기반 모델 구축을 위한 실현 가능한 기술 경로를 제공하는 데 있습니다.

핵심 내용

(a) 연구 배경 및 문제
현재 휴머노이드 로봇 동작 제어는 두 가지 핵심 과제에 직면해 있습니다: 첫째, 고품질 전신 동작 데이터를 효율적으로 획득하는 방법, 둘째, 로봇이 자연어 명령에 따라 실시간으로 다양한 동작을 생성하고 실행할 수 있도록 하는 방법입니다. 전통적인 방법은 수동으로 설계된 동작 라이브러리나 사전 프로그래밍된 궤적에 의존하여 복잡하고 변화무쌍한 실제 환경에 적응하기 어렵습니다. 반면 강화 학습 기반 방법은 일반화 능력을 갖추고 있지만 훈련 비용이 높고 실시간성을 보장하기 어렵습니다. TextOp는 이러한 병목 현상을 해결하기 위해 언어 명령을 고수준 의미 입력으로 사용하고, 원격 조작 또는 외골격으로 수집된 인간 시연 데이터를 결합하여 종단 간 동작 생성 및 제어 폐쇄 루프를 구축함으로써 수동 코딩 의존도를 낮추고 시스템의 유연성과 확장성을 향상시킵니다.

(b) 방법 또는 모델 프레임워크
TextOp의 핵심 프레임워크는 세 가지 주요 모듈로 구성됩니다: 데이터 수집 계층, 동작 생성 계층 및 제어 실행 계층. 데이터 수집 계층에서는 원격 조작 또는 외골격 장치를 통해 인간 조작 명령과 로봇 상태 데이터를 동기적으로 기록합니다. 동작 생성 계층은 확산 전략 또는 흐름 매칭 모델을 사용하여 언어 명령을 조건 신호로 삼고, 인간 시연 데이터로 구축된 다중 모드 동작 분포에서 연속적인 전신 궤적을 샘플링합니다. 제어 실행 계층은 전신 제어기(WBC) 또는 모델 예측 제어기(MPC)를 통해 생성된 궤적을 저수준 제어 목표로 변환하여 로봇 관절 모터를 구동해 안정적인 동작을 실현합니다. 전체 프로세스는 반복적으로 최적화 가능한 데이터 폐쇄 루프를 형성하여, 각 실행 후의 새로운 데이터가 생성 모델에 피드백되어 지속적으로 동작 품질을 향상시킵니다.

(c) 핵심 기술 혁신
TextOp의 기술 혁신은 주로 세 가지 측면에서 나타납니다: 첫째, 동작 생성을 조건부 확률 샘플링 문제로 간주하고, 확산 모델 또는 흐름 매칭의 생성 능력을 활용하여 다중 모드 동작 분포를 처리함으로써 전통적인 방법에서 단일 궤적 피팅으로 인한 모드 붕괴를 방지합니다. 둘째, 언어 명령과 동작 데이터 간의 정렬 메커니즘을 설계하여 텍스트 설명이 관절 각도, 발 궤적 등 저수준 동작 매개변수에 정확하게 매핑될 수 있도록 합니다. 마지막으로, 전신 제어기와 생성 모델을 융합하여 고수준 의미에서 저수준 제어로의 실시간 변환을 실현하며, 동작의 자연스러움을 보장하면서 로봇의 동적 균형과 추적 정밀도를 유지합니다. 이러한 혁신 덕분에 TextOp는 동작 다양성, 명령 응답 속도 및 실행 견고성에서 기존 방법보다 우수합니다.

(d) 실험/검증 또는 응용 가치
논문이 구체적인 실험 데이터를 공개하지는 않았지만, 프레임워크 설계로 미루어 볼 때 TextOp는 다양한 휴머노이드 로봇 플랫폼에 적용 가능하며, 특히 걷기, 회전, 잡기 등 다양한 동작 모드를 빠르게 전환해야 하는 시나리오에 적합합니다. 응용 측면에서 이 기술은 로봇 프로그래밍의 진입 장벽을 크게 낮출 수 있습니다: 작업자는 자연어로 명령을 내리기만 하면 시스템이 자동으로 해당 전신 동작을 생성하고 실행합니다. 또한 데이터 폐쇄 루프 특성 덕분에 로봇은 실제 배치에서 지속적으로 경험을 축적하여 점차 인간 동작의 유연성과 적응성에 근접할 수 있습니다. 향후 TextOp는 범용 행동 기반 모델의 핵심 구성 요소가 되어 휴머노이드 로봇의 서비스, 제조 및 구조 등 분야에서의 실용화를 촉진할 것으로 기대됩니다.
