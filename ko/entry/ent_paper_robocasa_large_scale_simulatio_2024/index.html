요약

RoboCasa: Large-Scale Simulation of Everyday Tasks for Generalist Robots is a 2024 work on simulation benchmark for humanoid robots, with open-source code available.

출처

  • RoboCasa: Large-Scale Simulation of Everyday Tasks for Generalist Robots (arXiv)
  • RoboCasa: Large-Scale Simulation of Everyday Tasks for Generalist Robots project page

개요

최근 인공지능(AI)의 발전은 주로 스케일링에 의해 추진되어 왔습니다. 로보틱스 분야에서는 대규모 로봇 데이터셋에 대한 접근 부족으로 인해 스케일링이 제한되고 있습니다. 우리는 현실적인 물리 시뮬레이션을 활용하여 로봇 학습 방법을 위한 환경, 작업 및 데이터셋을 확장할 것을 주장합니다. 우리는 일상 환경에서 범용 로봇을 훈련시키기 위한 대규모 시뮬레이션 프레임워크인 RoboCasa를 제시합니다. RoboCasa는 주방 환경에 초점을 맞춘 현실적이고 다양한 장면을 특징으로 합니다. 우리는 150개 이상의 객체 카테고리에 걸친 수천 개의 3D 자산과 수십 개의 상호작용 가능한 가구 및 가전제품을 제공합니다. 텍스트-3D 모델의 객체 자산과 텍스트-이미지 모델의 환경 텍스처와 같은 생성형 AI 도구를 통해 시뮬레이션의 현실성과 다양성을 강화합니다. 체계적인 평가를 위해 100개의 작업 세트를 설계했으며, 여기에는 대규모 언어 모델의 지침에 의해 생성된 복합 작업이 포함됩니다. 학습을 촉진하기 위해 고품질의 인간 시연을 제공하고 자동 궤적 생성 방법을 통합하여 최소한의 인간 부담으로 데이터셋을 대폭 확장합니다. 우리의 실험은 합성 생성 로봇 데이터를 대규모 모방 학습에 사용할 때 명확한 스케일링 추세를 보여주며, 실제 작업에서 시뮬레이션 데이터를 활용할 큰 가능성을 보여줍니다. 비디오 및 오픈소스 코드는 https://robocasa.ai/에서 확인할 수 있습니다.
