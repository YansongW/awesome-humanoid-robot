요약

arXiv:2606.31101v1 Announce Type: new Abstract: Bridging the sim-to-real gap is a core challenge in deploying learned manipulation policies. Sim-to-real learning is attractive because it can replace expensive real robot demonstrations with scalable synthetic data, yet world-action models have not previously been shown to transfer from simulation to real robotic manipulation. We study whether a world-action model can be trained from synthetic priors and deployed zero-shot in the real world. To this end, we build upon Cosmos Policy, a video diffusion model adapted for visuomotor control. We construct simulation environments with extensive domain randomization and generate demonstrations using the AnyTask motion planning pipeline. We evaluate our approach across object lifting, drawer opening, and pick-and-place tasks using ${\sim}800$ synthetic demonstrations per task and no real demonstrations. When deployed zero-shot on a Franka Robot, our policy attains a 35\% average success rate. To our knowledge, this represents the first successful sim-to-real transfer of a world-action model for robotic manipulation.

개요

시뮬레이션과 실제 환경 간의 차이를 극복하는 것은 학습된 조작 정책을 배포하는 데 있어 핵심적인 과제입니다. 시뮬레이션-실제 학습은 비용이 많이 드는 실제 로봇 시연을 확장 가능한 합성 데이터로 대체할 수 있어 매력적이지만, 월드-액션 모델이 시뮬레이션에서 실제 로봇 조작으로 전이된 사례는 이전에 보고된 바 없습니다. 우리는 월드-액션 모델이 합성 사전 지식으로부터 학습되어 실제 환경에서 제로샷으로 배포될 수 있는지 연구합니다. 이를 위해, 우리는 시각운동 제어에 맞게 조정된 비디오 확산 모델인 Cosmos Policy를 기반으로 구축합니다. 광범위한 도메인 무작위화를 적용한 시뮬레이션 환경을 구성하고, AnyTask 모션 플래닝 파이프라인을 사용하여 시연을 생성합니다. 우리는 객체 들어올리기, 서랍 열기, 집기-놓기 작업에 대해 각 작업당 \({\sim}800\)개의 합성 시연을 사용하고 실제 시연은 사용하지 않고 접근 방식을 평가합니다. Franka 로봇에 제로샷으로 배포했을 때, 우리의 정책은 35%의 평균 성공률을 달성했습니다. 우리가 아는 한, 이는 로봇 조작을 위한 월드-액션 모델의 최초의 성공적인 시뮬레이션-실제 전이를 나타냅니다.
