정책 병합을 통한 로봇 군단 학습

Robot Fleet Learning via Policy Merging

영역
AI 모델 및 알고리즘대량 생산소프트웨어 미들웨어
계층
지능 계층중위
검증
대기 중

요약

원시 데이터를 중앙 집중화하지 않고 이질적인 로컬 로봇 데이터로 훈련된 순환 신경망 정책을 통합하는 분산 정책 병합 방법 FLEET-MERGE를 제안하고 FLEET-TOOLS 도구 사용 벤치마크를 소개한다.

출처

  • Robot Fleet Learning via Policy Merging

개요

로봇 함대는 환경과 상호작용하며 생성되는 방대한 양의 이기종 스트리밍 데이터 사일로를 흡수하며, 이는 저장이나 전송이 용이한 수준을 훨씬 초과합니다. 동시에 로봇 팀은 다양한 환경에서의 이기종 경험을 통해 다양한 기술을 공동으로 습득해야 합니다. 함대 규모의 데이터를 전송하거나 중앙화하지 않고 이러한 함대 수준 학습을 어떻게 가능하게 할 수 있을까요? 본 논문에서는 이러한 분산 이기종 데이터셋으로부터의 정책 병합(PoMe)을 잠재적 해결책으로 조사합니다. 함대 환경에서 정책을 효율적으로 병합하기 위해, 순환 신경망으로 제어 정책을 매개변수화할 때 발생하는 순열 불변성을 고려한 분산 학습의 구현체인 FLEET-MERGE를 제안합니다. 우리는 FLEET-MERGE가 Meta-World 환경에서 50개 작업에 대해 훈련된 정책의 행동을 통합하며, 테스트 시점에 거의 모든 훈련 작업에서 우수한 성능을 보임을 입증합니다. 또한, 구성적이고 접촉이 많은 로봇 조작 작업에서 함대 정책 학습을 위한 새로운 로봇 도구 사용 벤치마크인 FLEET-TOOLS를 도입하여, 해당 벤치마크에서 FLEET-MERGE의 효용성을 검증합니다.
