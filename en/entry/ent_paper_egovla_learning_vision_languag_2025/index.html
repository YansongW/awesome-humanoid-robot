Summary

The implementation path of egofla is to first encode language instructions, camera images/multi-view observations, ontological states, and joint sequences into multimodal representations, which can be used to generate multimodal representations, then the whole-body trajectory/action sequence, end-effector/wrist-hand target were predicted with ACT/behavior clone imitation learning, VLA multimodal action model, latent variable/action token. The key point is to retain the semantic understanding of VLM, while increasing the robot state and action head, so as to avoid remaining in the language planning.

Overview

EgoVLA proposes a new paradigm for learning Vision-Language-Action (VLA) models from egocentric human videos, aiming to address the issue of reliance on expensive robot demonstration data in traditional robot imitation learning. The key contribution of this research lies in combining the semantic understanding capabilities of pre-trained Vision-Language Models (VLMs) with robot proprioceptive states and action prediction heads, enabling direct transfer from natural human behavior videos to whole-body robot action policies, thus avoiding the limitation of staying only at the language planning level. This work provides a feasible technical pathway for leveraging massive internet human video data to drive robot skill learning.

Content

Research Background and Problem
The core bottleneck in current robot learning is the scarcity and high cost of acquiring high-quality demonstration data. Traditional behavior cloning methods typically require human operators to teleoperate robots or use motion capture devices to collect data, which is not only inefficient but also struggles to cover diverse manipulation scenarios. Meanwhile, there is a vast amount of egocentric human daily activity videos (e.g., cooking, assembling, grasping) on the internet, rich in action-environment interaction information. However, converting these unstructured human videos into executable robot policies, especially preserving the high-level semantic understanding of vision-language models while generating precise low-level action sequences, remains an unresolved challenge.

Method or Model Framework
EgoVLA constructs an end-to-end multimodal learning framework, with a core process comprising three steps: First, language instructions, multi-view camera images, robot proprioceptive states (e.g., joint angles), and joint sequences from human videos are uniformly encoded into multimodal representations. Second, a Transformer-based architecture (e.g., ACT or behavior cloning variants) is used for temporal modeling of these representations, incorporating latent variables and action tokens to capture action stochasticity and multimodal distribution. Finally, a specially designed action prediction head outputs whole-body trajectories, end-effector poses, and fine-grained wrist-hand actions. The key design of this framework is to add a robot state encoding layer and an action decoding layer while retaining the pre-trained VLM weights, enabling the model to both understand semantic instructions like "pick up the cup" and generate specific joint angle sequences.

Key Technical Innovations
The technical innovations of EgoVLA are mainly reflected in three aspects: First, it proposes a "semantic preservation-action enhancement" model architecture, which freezes the VLM's semantic layers and only fine-tunes action-related modules, avoiding catastrophic forgetting and maintaining generalization to natural language instructions in zero-shot scenarios. Second, it designs a cross-domain alignment strategy from human first-person videos to robot action space, utilizing the mapping between human kinematics and robot kinematics to convert hand trajectories observed in videos into reference paths for the robot's end-effector. Third, it introduces a discretized representation method for action tokens, mapping the continuous action space into a finite set of tokens, thereby being compatible with the causal language modeling paradigm of VLMs and unifying language planning with action generation.

Experiments/Validation and Application Value
Although the paper abstract does not provide specific experimental data, based on its technical approach, it can be inferred that EgoVLA has significant advantages in typical tasks such as tabletop manipulation, object grasping, and tool use. Compared to traditional methods relying solely on robot demonstration data, this model can leverage richer scene variations and interaction patterns from human videos, thereby improving policy generalization. Additionally, due to retaining the VLM's semantic understanding capabilities, EgoVLA naturally supports the decomposition and execution of multi-step tasks via language instructions. In practical applications, this framework is expected to lower the barrier to robot skill acquisition, allowing non-expert users to teach robots new tasks by recording first-person videos, which holds significant value for scenarios such as home service and industrial assistance.
