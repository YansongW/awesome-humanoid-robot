Summary

The implementation path of TWIST2 is to first encode camera images/multi-view observations, human videos/motion capture trajectories, teleoperation/exoskeleton data into multimodal representations, and then use ACT/behavioral cloning imitation learning, VLA multimodal action model, and whole-body controller/WBC/MPC prediction to execute action commands. The key point is to preserve the semantic understanding of VLM while adding robot states and action heads to avoid staying only at language planning.

Overview

A core bottleneck for humanoid robots transitioning from laboratories to real-world applications is the scarcity of high-quality, diverse manipulation data. Existing data collection systems often struggle to balance scalability, portability, and data integrity, limiting the development of large-scale imitation learning and multimodal action models. To address this, the paper proposes the TWIST2 system, aiming to build a scalable, portable, and holistic data collection framework for humanoid robots. Its main contribution lies in designing a complete implementation pipeline from multi-source perception data encoding to executable action commands, and innovatively integrating the semantic understanding capabilities of vision-language models with robot states and action heads, thereby breaking through the limitation of traditional methods that remain at the level of language planning.

Content

In the current field of humanoid robot research, data-driven imitation learning and multimodal action models have become key technologies for enhancing robots' dexterous manipulation and whole-body coordination capabilities. However, existing data collection schemes generally face two major dilemmas: first, the data sources are singular, making it difficult to integrate multiple modalities such as camera images, human videos, motion capture trajectories, and teleoperation/exoskeleton data; second, the system architecture lacks scalability and portability, resulting in limited data scale and rigid collection scenarios. These issues severely constrain robots' ability to learn complex, generalizable behaviors from demonstrations, necessitating a systematic solution that can unify multimodal data representations and support efficient action generation.

The TWIST2 system proposes a hierarchical implementation path. First, the system uniformly encodes heterogeneous information from camera images/multi-view observations, human videos/motion capture trajectories, teleoperation/exoskeleton data, etc., into multimodal representations. The key to this step lies in establishing cross-modal alignment and fusion mechanisms, enabling demonstration data from different sources to be expressed within the same feature space. Subsequently, the system utilizes ACT (Action Chunking with Transformers) behavioral cloning imitation learning, VLA (Vision-Language-Action) multimodal action models, and whole-body controllers (WBC/MPC) predictions to transform multimodal representations into executable robot action commands. This end-to-end pipeline design from perception to planning ensures seamless integration between data collection and policy learning.

The core technological innovation of this method lies in the deep integration and extension of vision-language models (VLMs). Traditional approaches often use VLMs only for high-level semantic understanding or language planning, whereas TWIST2 enables VLMs to directly participate in low-level action generation by adding robot state and action heads. Specifically, the system retains the powerful semantic understanding capabilities of VLMs while extending them with specialized network branches for processing robot joint angles, end-effector poses, and other state information, as well as action heads that output continuous action sequences. This design avoids the disconnect of "planning without execution," allowing robots to both understand task semantics and generate fine-grained manipulation instructions.

In terms of experimental validation and application value, the TWIST2 system demonstrates its effectiveness across various manipulation tasks by constructing a large-scale, multimodal humanoid robot dataset. The system supports portable deployment, enabling rapid data collection in different environments (e.g., laboratories, homes, factories), significantly lowering the barrier to data acquisition. Additionally, due to its modular design, TWIST2 is compatible with various teleoperation devices and sensor configurations, offering flexible customization space for different research teams. The successful development of this system lays a solid data foundation for large-scale imitation learning, cross-scenario generalization, and human-robot collaborative manipulation of humanoid robots, potentially advancing humanoid robots from specific task execution to general manipulation capabilities.
