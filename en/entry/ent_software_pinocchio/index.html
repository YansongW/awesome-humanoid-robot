Definition and Positioning of Pinocchio

Pinocchio belongs to the software platform category. Its domain includes software middleware. Value chain level: intelligence layer. It is an open-source C++ library for efficient rigid-body dynamics, kinematics, and analytical derivative computation, widely used in robotics research and model-based control. The English name is Pinocchio. The Korean name is Pinocchio.
