Definition and Positioning of Linux RT-PREEMPT

Linux RT-PREEMPT belongs to the software platform type. Its domain includes software middleware. It is positioned at the intelligence layer of the value chain. It is a kernel patch set that makes most of the Linux kernel preemptible to provide deterministic low-latency real-time behavior, commonly used in robot control. Its English name is Linux RT-PREEMPT. Its Korean name is Linux RT-PREEMPT.
