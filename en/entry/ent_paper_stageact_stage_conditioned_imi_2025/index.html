Summary

StageACT mainly solves the data closed loop: using camera images/multi-view observation, ontology state and joint sequence, teleoperation/exoskeleton data to collect human operation and robot state, and then through ACT/behavioral cloning imitation learning, diffusion strategy/stream matching, VLM semantic planning/routing to trainable, reusable and executable action commands. The key point is to view action generation as a conditional generation problem and use diffusion or flow matching to sample executable trajectories in a multimodal action distribution.

Overview

Humanoid robots performing dexterous manipulation tasks in complex environments, such as opening doors, still face challenges of incomplete data loops and multimodal action distributions. Existing imitation learning methods often treat action generation as a single mapping problem, making it difficult to adapt to the dynamic changes in visual and proprioceptive states across different door-opening phases (e.g., grasping the handle, rotating, pulling). StageACT proposes a phase-conditioned imitation framework that redefines action generation as a conditional generation problem. By using diffusion or flow matching to sample executable trajectories from multimodal action distributions, it enhances the robustness and generalization of robot door opening.

Content

The humanoid robot door-opening task involves a complete closed loop from environmental perception to action execution, including real-time fusion of camera images/multi-view observations, proprioceptive states, and joint sequences. Existing methods typically rely on human operation data collected via teleoperation or exoskeletons, learning policies through behavior cloning (e.g., ACT). However, when facing different door-opening phases (e.g., initial positioning, force application for rotation, sustained pulling), a single policy model struggles to capture the unique action patterns and constraints of each phase. The core motivation of StageACT is to address this "data loop" problem: how to efficiently transform multimodal observations, human demonstrations, and robot states into trainable and reusable action commands while ensuring smooth transitions between phases.

The methodological framework of StageACT is based on a phase-conditioned generative model. It decomposes the entire door-opening task into several semantic phases (e.g., "approaching the handle," "grasping," "rotating," "pulling"), each corresponding to a conditioned action generation subproblem. Specifically, the system uses a Vision-Language Model (VLM) for semantic planning and routing, identifying the current phase from multi-view images and outputting phase labels as conditional variables. Subsequently, using diffusion policies or flow matching models, executable trajectories are sampled from multimodal action distributions given the phase condition, proprioceptive state, and joint sequences. This design enables the model to learn differentiated action priors for different phases, avoiding the ambiguity of a global policy.

Key technological innovations are reflected in three aspects: First, the phase-conditioning mechanism decomposes complex tasks into subproblems, reducing the multimodality of action distributions and allowing diffusion or flow matching models to more accurately fit the action patterns of each phase. Second, VLM semantic routing achieves end-to-end reasoning from visual observations to phase labels without requiring manual annotation of phase boundaries. Finally, through teleoperation/exoskeleton data collection, the system can simultaneously leverage human operation demonstrations and the robot's actual state, forming a closed-loop training data stream. These innovations collectively enhance the model's adaptability to visual changes during door opening (e.g., handle type, door angle) and physical interactions (e.g., force direction).

In terms of experimental validation, StageACT was tested on a real humanoid robot platform for door opening, covering various door types (sliding doors, revolving doors) and handle types. Results show that compared to baseline methods without phase conditioning (e.g., standard ACT or diffusion policies), StageACT achieves significant improvements in task success rate, action smoothness, and robustness to initial position perturbations. Additionally, the phase-conditioning mechanism allows the model to achieve comparable performance with fewer demonstration data, reducing data collection costs. This work provides a new paradigm for scene understanding and vision-guided control in mobile manipulation, and its phase decomposition approach can be extended to other long-horizon dexterous manipulation tasks.
