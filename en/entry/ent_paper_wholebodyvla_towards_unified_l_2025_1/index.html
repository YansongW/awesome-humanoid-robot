Summary

The implementation path of WholeBodyVLA is to first encode camera images/multi-view observations, teleoperation/exoskeleton data into multimodal representations, and then use the VLA multimodal action model to predict the whole-body trajectory/action sequence. The key point is to preserve the semantic understanding of VLM while adding robot states and action heads to avoid staying only at language planning.

Overview

Humanoid robots performing whole-body mobile manipulation tasks in complex environments face the challenge of deep integration between perception, planning, and control. Existing methods often confine Vision-Language Models (VLMs) to high-level semantic planning, making it difficult to directly generate fine-grained whole-body motion commands. To address this, this paper proposes the WholeBodyVLA framework, aiming to construct a unified latent Vision-Language-Action (VLA) model that achieves end-to-end prediction from multimodal perception to whole-body trajectory/action sequences. The main contribution of this work lies in preserving the powerful semantic understanding capabilities of VLMs while breaking through the limitations of language planning by introducing robot state encoding and action heads, enabling the model to directly output executable whole-body control commands.

Content

(a) Research Background and Problem: Whole-body loco-manipulation of humanoid robots requires high coordination among walking, balance, and fine-grained hand operations. Traditional hierarchical control architectures typically separate visual perception, semantic planning, and motion control, leading to information transmission loss and response delays. In recent years, Vision-Language-Action (VLA) models have shown potential in robotic manipulation, but existing work mostly focuses on fixed-base robotic arms or simple grasping tasks, making it difficult to handle the complexity of whole-body coordination in humanoid robots. The core problem is: how to enable VLA models not only to understand scene semantics but also to directly generate continuous action sequences containing joint angles, end-effector trajectories, and whole-body postures, rather than merely outputting task plans in linguistic form.

(b) Method or Model Framework: WholeBodyVLA proposes a unified latent space learning paradigm. Its implementation first maps camera images, multi-view observation data, and demonstration data from teleoperation or exoskeleton systems into a shared latent representation via a multimodal encoder. Subsequently, this representation is fed into a VLA multimodal action model, which predicts whole-body trajectories and action sequences in an autoregressive manner. The key design is that, while retaining the semantic understanding capabilities of pre-trained VLMs, the model additionally incorporates a robot state encoding branch and action prediction heads, ensuring that the output contains not only task intent but also motion commands directly usable for low-level control.

(c) Key Technical Innovations: The core innovation of this framework lies in the construction of a "unified latent space" and the joint learning of "semantics and actions." First, by aligning vision, language, and robot proprioceptive states into the same latent space, the model can implicitly learn the relationships between scene geometry, object properties, and motion constraints. Second, unlike methods that only output linguistic plans, WholeBodyVLA explicitly adds action heads to the model structure, allowing the semantic output of the VLM to be directly transformed into continuous joint trajectories or whole-body posture sequences. Third, using teleoperation data as supervision signals, the model learns whole-body coordination patterns from human demonstrations, thereby improving generalization ability and motion naturalness.

(d) Experiments/Validation and Application Value: Although the paper does not provide specific experimental data, it can be inferred from its technical approach that the framework is suitable for complex scenarios requiring whole-body coordination, such as humanoid robots carrying heavy objects in unstructured environments, navigating narrow passages, or performing assembly tasks. By deeply integrating VLM semantic understanding with whole-body motion control, WholeBodyVLA is expected to significantly reduce the abstraction level of task planning and minimize manually designed intermediate state machines or motion primitives. At the application level, this work provides a feasible technical path for the next generation of humanoid robots to achieve integrated "perception-understanding-action" control, particularly suitable for scenarios with high demands on whole-body coordination, such as home services, industrial inspection, and disaster rescue.
