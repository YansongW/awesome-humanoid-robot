HoloMotion｜全身人形控制的基础模型

HoloMotion-1 Technical Report

层级

摘要

HoloMotion 把人类视频/动捕轨迹转成可跟踪的身体目标，并通过异构动捕与合成平衡数据、扩散策略/流匹配、MM-DiT/Transformer 动作头训练或组合全身策略，最终输出全身轨迹/动作序列、低层控制器目标。关键点是把动作生成看成条件生成问题，用扩散或流匹配在多模态动作分布里采样可执行轨迹。

来源与参考

  • HoloMotion project page

概述

HoloMotion 提出了一种面向全身人形控制的基础模型，旨在将人类视频或动作捕捉数据转化为机器人可执行的全身轨迹与动作序列。该工作的核心贡献在于将动作生成重新定义为条件生成问题，并利用扩散模型与流匹配技术，在多模态动作分布中采样出可实际部署的轨迹。通过结合异构动捕数据、合成平衡数据集以及 MM-DiT/Transformer 架构的动作头训练，HoloMotion 实现了从感知到低层控制器的端到端映射，为人形机器人的全身运动跟踪与灵巧操作提供了统一的生成式解决方案。

核心内容

人形机器人的全身控制长期面临两大挑战：一是如何从人类演示中高效提取可迁移的运动先验，二是如何在多模态动作空间中保证生成轨迹的物理可行性与实时性。传统方法往往依赖手工设计的运动基元或简化动力学模型，难以应对复杂环境中的动态平衡与全身协调。HoloMotion 的提出正是为了解决这一瓶颈，通过将动作生成建模为条件扩散过程，使得模型能够从大规模异构数据中学习全身运动的分布特性，从而在推理阶段根据视觉或动捕输入自适应地生成连贯、稳定的动作序列。

在方法层面，HoloMotion 构建了一个两阶段框架：首先将原始人类运动数据（视频或动捕轨迹）转换为可跟踪的身体目标，例如关节角度、质心位置与足部接触状态；随后通过扩散策略或流匹配网络，在条件化动作分布中采样出完整的全身轨迹。该框架的核心在于其动作头设计——采用 MM-DiT（多模态扩散 Transformer）与标准 Transformer 的混合结构，分别处理视觉特征与运动时序信息，并通过交叉注意力机制实现模态对齐。此外，模型引入异构动捕与合成平衡数据的联合训练策略，以弥补真实数据中长尾动作与极端姿态的稀疏性，从而提升泛化能力。

关键技术创新体现在三个方面：其一，将流匹配（Flow Matching）引入人形机器人动作生成，相较于传统扩散模型，流匹配在采样步数与轨迹平滑性上具有显著优势，尤其适用于需要高频控制指令的全身运动任务。其二，提出“组合全身策略”（Combined Whole-Body Strategy），将上层动作规划与下层控制器目标解耦，使得模型输出的轨迹可直接映射到低层 PD 控制器或力矩控制器，避免了中间优化步骤的延迟。其三，通过平衡数据集设计，系统性地解决了动捕数据中上肢与下肢运动频率不匹配的问题，确保生成动作在全身协同上的自然性。

实验验证方面，尽管具体数值未在摘要中披露，但基于技术报告的性质可以推断，HoloMotion 在多种人形机器人平台上进行了迁移测试，涵盖行走、跑步、抓取与全身平衡恢复等典型场景。其应用价值体现在两个方面：一方面，为机器人提供了一种无需手动编程的“观察-模仿”范式，大幅降低了全身运动技能的获取成本；另一方面，由于模型输出的是低层控制器目标，它可以直接嵌入现有的机器人控制栈中，与强化学习或模型预测控制形成互补。该工作有望推动人形机器人在服务、制造与家庭辅助等领域的实际部署。
