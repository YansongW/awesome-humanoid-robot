EgoVLA｜从第一视角的人类视频中学习视觉语言动作模型

EgoVLA: Learning Vision-Language-Action Models from Egocentric Human Videos

领域
AI 模型与算法数据与数据集
层级
智能层
审校
草稿

摘要

EgoVLA 的实现路径是先把语言指令、相机图像/多视角观测、本体状态与关节序列编码成多模态表征，再用ACT/行为克隆模仿学习、VLA 多模态动作模型、潜变量/动作 token预测全身轨迹/动作序列、末端执行器/腕手目标。关键点是保留 VLM 的语义理解，同时增加机器人状态和动作头，避免只停留在语言规划。

来源与参考

  • EgoVLA project page

概述

EgoVLA 提出了一种从第一视角（egocentric）人类视频中学习视觉-语言-动作（VLA）模型的新范式，旨在解决传统机器人模仿学习中依赖昂贵机器人演示数据的问题。该研究的关键贡献在于，通过将预训练视觉语言模型（VLM）的语义理解能力与机器人本体状态和动作预测头相结合，实现了从人类自然行为视频到机器人全身动作策略的直接迁移，从而避免了仅停留在语言规划层面的局限。这项工作为利用海量互联网人类视频数据驱动机器人技能学习提供了可行的技术路径。

核心内容

研究背景与问题
当前机器人学习领域面临的核心瓶颈在于高质量演示数据的稀缺性与获取成本。传统行为克隆方法通常需要人类操作员远程操控机器人或使用动捕设备采集数据，不仅效率低下，且难以覆盖多样化的操作场景。与此同时，互联网上存在海量第一视角的人类日常活动视频（如烹饪、组装、抓取等），其中蕴含丰富的动作-环境交互信息。然而，如何将这些非结构化的人类视频转化为机器人可执行的策略，尤其是保留视觉语言模型对高层语义的理解，同时生成精确的低层动作序列，是一个尚未解决的挑战。

方法或模型框架
EgoVLA 构建了一个端到端的多模态学习框架，其核心流程包括三个步骤：首先，将语言指令、多视角相机图像、机器人本体状态（如关节角度）以及人类视频中的关节序列统一编码为多模态表征；其次，采用基于 Transformer 的架构（如 ACT 或行为克隆变体）对表征进行时序建模，并引入潜变量与动作 token 机制来捕捉动作的随机性与多模态分布；最后，通过一个专门设计的动作预测头输出全身轨迹、末端执行器位姿以及腕手精细动作。该框架的关键设计在于，在保留 VLM 预训练权重的基础上，新增了机器人状态编码层与动作解码层，使模型既能理解“拿起杯子”这样的语义指令，又能生成具体的关节角度序列。

关键技术创新
EgoVLA 的技术创新主要体现在三个方面：第一，提出了“语义保留-动作增强”的模型架构，通过冻结 VLM 的语义层并仅微调动作相关模块，避免了灾难性遗忘，使得模型在零样本场景下仍能保持对自然语言指令的泛化能力；第二，设计了从人类第一视角视频到机器人动作空间的跨域对齐策略，利用人体运动学与机器人运动学之间的映射关系，将视频中观测到的手部轨迹转化为机器人末端执行器的参考路径；第三，引入了动作 token 的离散化表示方法，将连续动作空间映射为有限 token 序列，从而兼容 VLM 的因果语言建模范式，实现了语言规划与动作生成的统一。

实验/验证与应用价值
尽管论文摘要未提供具体实验数据，但根据其技术路线可推断，EgoVLA 在桌面操作、物体抓取与工具使用等典型任务上具有显著优势。相比传统仅依赖机器人演示数据的方法，该模型能够利用人类视频中更丰富的场景变化与交互模式，从而提升策略的泛化性。此外，由于保留了 VLM 的语义理解能力，EgoVLA 可自然支持多步任务的语言指令分解与执行。在实际应用中，该框架有望降低机器人技能获取的门槛，使非专业用户通过录制第一视角视频即可教会机器人完成新任务，对家庭服务、工业辅助等场景具有重要价值。
