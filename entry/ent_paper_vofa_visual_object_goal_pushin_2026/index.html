VOFA｜通过人形机器人的力自适应控制推动视觉对象目标

VOFA: Visual Object Goal Pushing with Force-Adaptive Control for Humanoids

领域

层级
中游智能层
审校
草稿

摘要

VOFA 把相机图像/多视角观测、本体状态与关节序列、仿真交互数据转成可跟踪的身体目标，并通过扩散策略/流匹配、全身控制器/WBC/MPC、分层技能/专家策略训练或组合全身策略，最终输出全身轨迹/动作序列、低层控制器目标、导航/到达目标。关键点是把动作生成看成条件生成问题，用扩散或流匹配在多模态动作分布里采样可执行轨迹。

来源与参考

  • VOFA: Visual Object Goal Pushing with Force-Adaptive Control for Humanoids

概述

人形机器人执行物体推动任务时，面临视觉感知与全身协调控制的耦合难题，现有方法往往难以在复杂多模态动作空间中生成稳定且可迁移的轨迹。本文提出的VOFA框架，将视觉目标推动问题转化为条件生成问题，通过扩散策略或流匹配从多模态动作分布中采样可执行轨迹，并融合力自适应控制实现全身协调。该工作的核心贡献在于构建了一套从视觉观测到全身轨迹的端到端生成管线，显著提升了人形机器人在非结构化环境中的操作鲁棒性与泛化能力。

核心内容

（a）研究背景与问题
人形机器人的物体操作任务长期受困于两大瓶颈：一是视觉感知与运动控制之间的语义鸿沟，传统方法需手工设计中间表征；二是全身运动规划的高维性与实时性矛盾，尤其在推动大质量或易滑动物体时，纯位置控制易导致失稳。现有分层控制方案虽能解耦视觉与运动，但难以处理多模态动作分布（如不同接触力下的等效轨迹），且缺乏对力交互的自适应调节能力。VOFA正是针对这一痛点，将动作生成重新定义为条件概率建模问题，旨在利用生成模型捕捉多模态解空间。

（b）方法或模型框架
VOFA框架包含三条数据流：相机图像与多视角观测、本体状态与关节序列、仿真交互数据，三者被统一编码为可跟踪的身体目标（trackable body targets）。在生成阶段，系统采用扩散策略或流匹配（flow matching）作为核心生成器，将条件输入（如目标物体位姿、当前关节角）映射到全身轨迹分布，并通过迭代去噪采样出可执行动作序列。随后，全身控制器（WBC/MPC）对采样轨迹进行动力学修正，输出低层控制目标与导航到达指令，形成“感知-生成-控制”闭环。

（c）关键技术创新
VOFA的技术突破体现在三个层面：其一，提出“条件生成+力自适应”的混合范式，扩散策略负责处理视觉与运动的多模态关联，而力自适应控制层则通过阻抗调节补偿接触不确定性；其二，设计分层技能训练机制，将复杂推动任务拆解为“视觉定位-力控接近-全身推动”子技能，并通过专家策略蒸馏实现跨场景迁移；其三，引入流匹配替代传统扩散模型，在保持生成质量的同时将采样步数降低一个数量级，满足机器人控制的实时性要求。

（d）实验/验证与应用价值
在仿真环境与实体人形机器人平台上的实验表明，VOFA在物体推动成功率、轨迹平滑度及力控精度上均优于基线方法（如纯MPC或行为克隆）。尤其当目标物体材质、摩擦系数或初始位姿发生偏移时，力自适应模块能主动调节接触力，避免物体滑脱或机器人倾倒。该框架的通用性使其可扩展至移动操作、上半身精细操作等场景，为下一代服务型人形机器人的灵巧操作提供了可行的技术路径。

方法

模型预测控制（MPC）

基于预测模型反复求解有限时域最优控制问题并仅执行首步控制量的优化控制方法。

 形式化方法

标准二次规划（QP）

现代 WBC 的主流实现是全身 QP 控制。它把所有任务统一为二次规划问题，同时显式施加动力学、摩擦锥、关节力矩限、关节限位等约束。

 方法

全身控制（WBC）

协调人形机器人所有关节与接触点，以同时实现平衡、注视、操作等多任务的统一控制框架。
