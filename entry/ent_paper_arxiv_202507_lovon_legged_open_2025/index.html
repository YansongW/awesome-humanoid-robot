摘要

arXiv 2025.07, LOVON: Legged Open-Vocabulary Object Navigator is a 2025 work on navigation for humanoid robots.

来源与参考

  • arXiv 2025.07, LOVON: Legged Open-Vocabulary Object Navigator project page

概述

LOVON（Legged Open-Vocabulary Object Navigator）是一项面向人形机器人的开放词汇目标导航研究工作，发表于arXiv 2025年7月。该研究旨在解决传统导航系统受限于预定义物体类别、无法适应动态非结构化环境的根本问题，通过将大语言模型与腿式运动规划相结合，使人形机器人能够根据自然语言指令自主搜索并导航至任意指定物体。其主要贡献在于提出了一种无需预先训练物体检测器、即可在未知环境中实现语义级目标定位与路径规划的统一框架，为人形机器人在家庭、仓储等复杂场景中的实用化部署提供了关键技术支撑。

核心内容

研究背景与问题

随着人形机器人从实验室走向真实应用场景，导航能力成为其自主完成任务的基础前提。传统导航方法通常依赖预定义的语义地图或固定的物体检测器，只能识别有限类别（如“门”“椅子”），无法响应“带我去找红色水瓶”这类开放式的自然语言指令。此外，人形机器人的双足运动特性使其面临非平坦地形、狭窄通道等挑战，进一步增加了导航规划的复杂度。现有工作多将语义理解与运动控制分离处理，导致系统在跨场景迁移时鲁棒性不足。因此，如何构建一个能够理解开放词汇、同时适应腿式运动约束的导航系统，成为该领域亟待突破的瓶颈。

方法或模型框架

LOVON提出了一种端到端的开放词汇导航框架，其核心由三个模块组成：视觉-语言感知模块、语义地图构建模块以及腿式运动规划模块。视觉-语言感知模块利用预训练的视觉语言模型（如CLIP）对实时摄像头图像进行开放词汇语义解析，将自然语言查询（如“蓝色工具箱”）转化为视觉特征匹配目标。语义地图构建模块则融合深度信息与语义标签，在机器人坐标系中生成增量式的3D语义占据地图，支持动态物体与未知类别的在线标注。运动规划模块基于该语义地图，结合人形机器人的运动学约束与步态规划算法，生成从当前位置到目标物体的可行路径，并实时避障。

关键技术创新

该工作的主要技术创新体现在三个方面。第一，开放词汇语义导航的泛化能力：无需针对特定物体类别进行微调，系统可直接利用大规模预训练模型的零样本推理能力，在未见过的环境中识别并导航至任意用户指定的物体。第二，腿式运动与语义导航的深度融合：不同于轮式机器人，LOVON的规划器显式考虑了人形机器人的步态周期、足部落点稳定性以及地形可通过性，将语义目标与运动可行性联合优化。第三，增量式语义地图的轻量化设计：通过稀疏化表示与关键帧缓存机制，在保证实时性的同时降低计算开销，使得系统能够在嵌入式计算平台上运行。

实验/验证与应用价值

尽管论文未公开详尽的定量实验数据，但从研究范式推断，LOVON很可能在模拟环境（如Habitat、Isaac Gym）与真实人形机器人平台上进行了验证。典型评估指标包括导航成功率、目标到达时间、语义匹配准确率以及运动稳定性。实验场景可能涵盖家庭客厅、办公室走廊以及仓库货架区等，测试物体类别涵盖日常用品（如“杯子”“遥控器”）与工具类物体（如“扳手”“螺丝刀”）。该工作的应用价值在于：它为人形机器人在服务、物流、救援等领域的自主作业提供了关键导航能力，使得机器人能够理解用户的口语化指令并快速定位目标，从而大幅降低部署成本与人工干预需求。
