学习人形机器人视觉驱动的反应式足球技能

Learning Agile Striker Skills for Humanoid Soccer Robots from Noisy Sensory Input

层级

摘要

这篇工作先从相机图像/多视角观测、本体状态与关节序列、仿真交互数据恢复场景、目标或运动表征，再用教师-学生知识迁移、PPO/RL 策略训练、全身控制器/WBC/MPC生成可执行动作命令。关键点是用特权信息训练教师策略，再把能力蒸馏到只能使用部署观测的学生策略。

来源与参考

  • 学习人形机器人视觉驱动的反应式足球技能 project page

概述

本文针对人形机器人在动态对抗环境中执行敏捷足球技能所面临的感知与控制挑战，提出了一种基于视觉驱动的反应式技能学习框架。研究的主要贡献在于，通过构建教师-学生知识蒸馏架构，利用特权信息训练高精度教师策略，再将其能力迁移至仅依赖部署观测（如噪声相机图像）的学生策略，从而在仿真环境中实现了从视觉输入到全身运动命令的端到端映射。该工作有效弥合了仿真训练与真实部署之间的感知鸿沟，为人形机器人在复杂动态场景下的自主决策与敏捷控制提供了新的技术路径。

核心内容

人形机器人足球比赛要求机器人在高速运动、强对抗和部分可观测条件下，同时完成目标追踪、避障、控球与射门等复杂任务。传统方法通常依赖精确的状态估计和手工设计的运动规划器，但在实际部署中，传感器噪声、延迟和模型误差会显著降低系统鲁棒性。现有强化学习方法虽能学习端到端策略，却常因高维视觉输入与低维控制命令之间的巨大语义鸿沟而难以收敛。因此，如何从噪声多模态感知数据中提取有效运动表征，并生成敏捷且鲁棒的执行动作，成为该领域的关键难题。

该研究提出的框架包含两个核心阶段：教师策略训练与学生策略蒸馏。在教师阶段，系统利用特权信息（如精确的物体位置、速度、接触状态等）作为输入，通过近端策略优化（PPO）算法训练一个高性能的足球技能策略。教师策略能够基于理想化的状态表征规划出最优的全身运动，包括步态切换、躯干姿态调整与踢球动作的时序协调。随后，在蒸馏阶段，学生策略仅能访问部署环境下的真实观测，包括单目或多视角相机图像、本体状态（关节角度、IMU数据）以及关节序列历史。通过最小化教师与学生策略在动作分布上的KL散度，并结合行为克隆与强化学习联合训练，学生策略逐步学会从噪声视觉输入中推断出与教师策略等效的运动意图。

该方法的关键技术创新体现在三个方面。其一，提出了一种多模态感知融合架构，将视觉特征、本体状态与时间序列信息通过Transformer编码器进行联合建模，从而在特征层面实现场景理解与运动记忆的融合。其二，设计了分层全身控制器（WBC）作为策略输出层，将高层动作命令（如目标足部位置、躯干朝向）分解为关节力矩指令，确保运动平滑且满足动力学约束。其三，引入接触规划（contact planning）与移动操作（mobile manipulation）联合优化机制，使机器人能在奔跑过程中动态调整支撑脚与踢球脚的时序，实现类似人类球员的“跑动中射门”技能。

在实验验证方面，研究团队在基于物理引擎的高保真仿真环境中构建了多人形机器人对抗场景。教师策略在拥有完整状态信息的条件下，射门成功率超过95%，并能适应不同球速与防守者轨迹。经过蒸馏后的学生策略，在仅依赖单目RGB图像和本体感知的条件下，射门成功率仍保持在85%以上，且对相机噪声、光照变化和关节延迟表现出良好的鲁棒性。该框架不仅适用于足球技能，其“特权训练-蒸馏部署”范式还可推广至其他需要敏捷反应的人形机器人任务，如救援、仓储搬运等场景，具有重要的工程应用价值。
