附录 B 关键数据集清单

本附录由 scripts/build_appendices.py 从知识图谱数据自动生成，随 KG 更新而更新。

知识图谱收录的 11 个关键数据集实体，涵盖遥操作演示、第一视角视频、动作捕捉与大规模跨具身数据。

名称 英文名 简介
AMASS 动作数据集 AMASS Motion Dataset 大规模干净的动作捕捉数据集，包含人体形状和姿态，广泛用于人形机器人仿真预训练。
ARIO 全机器人合一数据集 ARIO Dataset 统一数据标准，约300万条片段，融合真实、仿真和转换的机器人数据来源。
BridgeData V2 BridgeData V2 广泛用于模仿学习的带语言指令的桌面操作数据集。
DROID 机器人操作数据集 DROID 一个分布式、多样化的真实世界机器人操作数据集，跨多个实验室和环境收集，旨在提升视觉与环境鲁棒性。
Ego4D 第一视角视频数据集 Ego4D Egocentric Video Dataset 3000多小时第一视角视频，用于第一人称技能和交互理解。
Epic-Kitchens 数据集 Epic-Kitchens Dataset 第一视角厨房交互视频数据集，广泛用于操作与活动理解。
HuDOR HuDOR 一种用于采集人手动作数据以训练灵巧机器人手模仿学习策略的人类到机器人演示数据集或遥操作框架。
HumanPlus Shadowing数据集 HumanPlus Shadowing Dataset 通过人形机器人跟随人体运动采集的数据集，用于训练全身模仿与遥操作策略。
OmniAction 数据集 OmniAction Dataset 一个用于主动式机器人操作的大规模多模态数据集，包含 14.
Open X-Embodiment 数据集 Open X-Embodiment Dataset 概述 一个大规模聚合机器人学习数据集，整合来自多种真实机器人平台和机构的演示数据，广泛用于跨具身 VLA 预训练。
RH20T 操作数据集 RH20T Manipulation Dataset 约11万条接触丰富的操作序列，包含视觉、力觉、音频和人类演示对。
