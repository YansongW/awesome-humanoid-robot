附录 C 软件与仿真平台清单

本附录由 scripts/build_appendices.py 从知识图谱数据自动生成，随 KG 更新而更新。

知识图谱收录的 19 个软件与仿真平台实体，覆盖中间件、运动规划、动力学库、物理引擎与大规模并行仿真。

名称 英文名 简介
Drake 系统工具箱 Drake Systems Toolbox MIT面向基于优化控制与分析的系统建模工具箱。
Gazebo Gazebo 提供物理引擎、传感器模型与场景编辑功能的开源三维机器人仿真器，用于算法开发与测试。
Genesis 生成式物理引擎 Genesis Generative Physics Engine 面向机器人等领域的新兴生成式通用物理引擎。
HumanoidVerse HumanoidVerse 面向Sim-to-Real人形机器人学习的多仿真器训练框架。
Isaac Gym基准 Isaac Gym Benchmarks 基于NVIDIA Isaac Gym构建的GPU加速强化学习基准集合，用于高吞吐量机器人策略训练与评估。
LeRobot LeRobot 覆盖数据集格式、遥操作、训练和部署的端到端PyTorch库。
Linux RT-PREEMPT Linux RT-PREEMPT 使Linux内核大部分可抢占以提供确定性低延迟实时行为的内核补丁集，常用于机器人控制。
MJCF 仿真格式 MJCF Simulation Format MuJoCo的XML建模格式，用于描述具有接触、执行器和传感器的铰接刚体系统。
ManiSkill3 ManiSkill3 面向具身AI的GPU并行机器人仿真与渲染基准。
MoveIt 运动规划 MoveIt Motion Planning MoveIt 是 ROS 生态中常用的运动规划框架，集成 OMPL 等规划器、逆运动学求解与碰撞检测，广泛用于机械臂及人形机器人的全身运动规划。
MuJoCo 物理引擎 MuJoCo Physics Engine 具有丰富接触动力学的高保真物理引擎，广泛用于人形控制研究。
NVIDIA Isaac Lab NVIDIA Isaac Lab 基于Isaac Sim的模块化学习框架，用于大规模机器人策略训练。
NVIDIA Isaac Sim NVIDIA Isaac Sim GPU加速的逼真机器人仿真器，支撑GR00T合成数据管道。
OMPL Open Motion Planning Library (OMPL) 包含RRT、PRM、BIT等先进基于采样运动规划算法的开源C++库。
Pinocchio Pinocchio 用于高效刚体动力学、运动学与解析导数计算的开源C++库，广泛应用于机器人研究与基于模型的控制。
QNX QNX 通用操作系统（如标准 Linux）并非为硬实时设计。
ROS 2 中间件 ROS 2 Middleware 基于DDS的发布/订阅和实时支持的事实标准机器人中间件。
URDF 机器人描述格式 URDF Robot Description Format 用于仿真和控制的描述机器人连杆、关节、惯性和几何的XML格式。
机群管理平台 Fleet Management Platform 跨多台已部署人形机器人进行任务编排、充电、健康监控和分析的云平台。
