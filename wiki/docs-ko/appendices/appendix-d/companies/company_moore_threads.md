## 회사 소개

Moore Threads는 풀 기능 GPU 개발에 주력하며, 그래픽 렌더링, AI 컴퓨팅 및 비디오 코덱 기능을 제공하고 국산 GPU 소프트웨어 및 하드웨어 생태계를 구축합니다.

Moore Threads는 중국의 대표적인 풀 기능 GPU 기업으로, MTT S 시리즈 데스크탑 및 데이터센터 GPU를 출시하여 DirectX, Vulkan, OpenGL, CUDA 호환 및 AI 프레임워크를 지원합니다. 회사는 그래픽과 컴퓨팅의 통합을 강조하며, 클라우드 게이밍, AI 추론, 디지털 트윈 및 메타버스를 위한 컴퓨팅 파워를 제공합니다.

## 대표 제품

### Moore Threads MTT S80

> Moore Threads MTT S80: [공식 자료](https://www.moorethreads.com)를 방문하여 확인하세요.

| 사양 항목 | 수치 | 비고/출처 |
|--------|------|-----------|
| 아키텍처 | MUSA (Moore Threads Unified System Architecture) | Moore Threads 공식 홈페이지 |
| 공정 | 미공개 | - |
| 비디오 메모리 | 16 GB GDDR6 | Moore Threads 공식 홈페이지 |
| 인터페이스 | PCIe Gen5 | Moore Threads 공식 홈페이지 |
| 그래픽 API | DirectX, Vulkan, OpenGL | Moore Threads 공식 홈페이지 |
| AI 컴퓨팅 성능 | 미공개 | - |
| 비디오 코덱 | AV1, H.265, H.264 지원 | Moore Threads 공식 홈페이지 |
| 가격 | 약 2999 RMB (참고 가격) | 공개 보도 |

**기술적 특징**: 국산 게임/그래픽 GPU, PCIe Gen5, 주요 그래픽 API 및 비디오 코덱 지원, MUSA 소프트웨어 생태계.

**적용 시나리오**: 데스크탑 게임, 그래픽 디자인, 비디오 편집, 경량 AI 추론.

## 공급망 위치

- **상류 핵심 부품/재료**: 웨이퍼 파운드리, GDDR/HBM 메모리, 패키징 및 테스트, EDA, 그래픽/컴퓨팅 IP, 서버 제조사.
- **하류 고객/적용 시나리오**: 게이머, 디자인 스튜디오, 클라우드 서비스 제공업체, 지능형 컴퓨팅 센터, AIGC 기업.
- **주요 경쟁사/대상**: NVIDIA GeForce/RTX, AMD Radeon, Intel Arc; 국산 대응 기업: Biren Technology, Muxi.
