## 회사 소개

국산 RV 감속기 선두 기업의 자회사 환동테크는 정밀 기어 및 로봇 감속기에 주력하고 있습니다.

双环传动은 자회사 환동테크를 통해 RV 감속기, 하모닉 감속기 및 유성 감속기를 공급하며, 3–1000 kg 부하 로봇에 필요한 정밀 감속기를 포괄합니다. 이미 宇树科技 등 주요 로봇 기업에 대량 납품하고 있습니다.

## 대표 제품

### SHPR-40E-121 RV 감속기 / SHPR-40E-121 RV Reducer

> SHPR-40E-121 RV 감속기: [공식 자료](http://www.shuanghuandrive.com)를 방문하여 확인하십시오.

| 사양 항목 | 수치 | 비고/출처 |
|--------|------|-----------|
| 모델 | SHPR-40E-121 | 가이스웨이 제품 페이지 |
| 감속비 | 121:1 | 가이스웨이 제품 페이지 |
| 정격 토크 | 412 N·m | 가이스웨이 제품 페이지 |
| 백래시 | ≤ 1 arcmin | 가이스웨이 제품 페이지 |
| 특징 | 메인 베어링 내장, 강한 비틀림 강성, 콤팩트한 체적 | 가이스웨이 제품 페이지 |
| 적용 분야 | 산업용 로봇 관절, 용접 포지셔너, 회전 테이블 | 가이스웨이 제품 페이지 |
| 중량 | 미공개 | - |
| 최대 입력 회전 속도 | 미공개 | - |

**기술적 특징**: 국산 RV 감속기의 대표 모델로, 성능이 Nabtesco 동급 제품에 근접하며 가성비가 높습니다.

**적용 시나리오**: 6축 산업용 로봇 J1–J3, 휴머노이드 로봇 허리/엉덩이, 중하중 포지셔너.

### SHPR-20E RV 감속기 / SHPR-20E RV Reducer

> SHPR-20E RV 감속기: [공식 자료](http://www.shuanghuandrive.com)를 방문하여 확인하십시오.

| 사양 항목 | 수치 | 비고/출처 |
|--------|------|-----------|
| 시리즈 | SHPR-E | 가이스웨이 제품 페이지 |
| 감속비 | 57/81/105/121/141/161:1 | Nabtesco RV-E 호환 시리즈 |
| 정격 토크 | 미공개 | - |
| 백래시 | ≤ 1 arcmin | 가이스웨이 제품 페이지 |
| 구조 | 2단 사이클로이드 유성 + 내장 메인 베어링 | 가이스웨이 제품 페이지 |
| 특징 | 고강성, 내충격성, 대감속비 | 가이스웨이 제품 페이지 |

**기술적 특징**: 중소형 RV 감속기로, 중소 부하 로봇 관절 및 정밀 회전 테이블에 적합합니다.

**적용 시나리오**: SCARA, 협동 로봇, 휴머노이드 로봇 다리 관절.
