## 회사 소개

상탕과학기술은 컴퓨터 비전과 멀티모달 대형 모델을 핵심으로, SenseCore 대형 장치와 일일신 대형 모델을 통해 구현 지능과 로봇에 인식, 이해 및 의사 결정 능력을 제공합니다.

상탕은 중국 선도적인 AI 소프트웨어 회사로, 사업은 스마트 비즈니스, 스마트 시티, 스마트 라이프 및 스마트 자동차를 포괄합니다. SenseCore 대형 장치는 연산력, 알고리즘 및 데이터 폐쇄 루프를 제공합니다. 일일신(SenseNova) 대형 모델 시스템은 언어, 시각, 멀티모달 및 코드 생성을 포괄합니다. 상탕은 또한 로봇/구현 지능을 위한 인식 및 상호 작용 솔루션을 출시했습니다.

> 상탕 일일신 / 구현 멀티모달 모델: [공식 자료](https://www.sensetime.com)를 방문하여 확인하세요.

| 사양 항목 | 수치 | 비고/출처 |
|--------|------|-----------|
| 모델 시리즈 | 일일신 SenseNova (언어/시각/멀티모달/구현) | 상탕 공식 사이트 |
| 매개변수 규모 | 수십억에서 천억 수준 (다양한 사양) | 상탕 공개 자료 |
| 능력 | 이미지 이해, 비디오 분석, 자연어 상호 작용, 작업 계획 | 상탕 공개 자료 |
| 구현 능력 | 시각 언어 동작(VLA) 지원, 환경 이해, 명령 추종 | 공개 보도 |
| 배포 | 클라우드 API / 사설화 / 엣지 적응 | 상탕 공식 사이트 |
| 훈련 연산력 | SenseCore 대형 장치 | 상탕 공식 사이트 |
| 인터페이스 | 상탕 API / 기업 맞춤형 | 상탕 공식 사이트 |
| 가격 | 호출량/사설화 배포에 따라 과금 | 상탕 공식 사이트 |

**기술 하이라이트**: 멀티모달 통합 아키텍처, 강력한 중국어 시각 이해, 구현 지능을 위한 동작 계획 및 장면 이해, SenseCore 훈련 추론 일체화.

**응용 시나리오**: 인간형 로봇 두뇌, 스마트 고객 서비스, 콘텐츠 생성, 산업 품질 검사, 자율 주행 인식.
