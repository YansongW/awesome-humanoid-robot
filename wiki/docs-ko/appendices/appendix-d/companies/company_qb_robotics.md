## 회사 소개

소프트 로봇 기술 기반의 인간형 로봇 핸드 개발사.

qb Soft Robotics(qbrobotics)는 이탈리아 피사에 본사를 두고 있으며, 소프트 로봇과 생체 모방 5핑거 로봇 핸드에 특화되어 있습니다. qb SoftHand 시리즈는 단일 모터 텐던 구동과 소프트 조인트 협력을 통해 인간과 유사한 파지 및 높은 환경 적응성을 구현하며, 자체 리셋, 높은 하중 대 중량비, 플러그 앤 플레이 등의 특징을 가지고 있어 연구, 교육, 협동 로봇 및 인간-로봇 상호작용 분야에서 널리 사용됩니다.

## 대표 제품

### qb SoftHand 5핑거 소프트 로봇 핸드 / qb SoftHand 5-Finger Soft Robotic Hand

> qb SoftHand 5핑거 소프트 로봇 핸드: [공식 자료](https://qbrobotics.com)를 방문하여 확인하세요.

| 사양 항목 | 수치 | 비고/출처 |
|--------|------|-----------|
| 크기 | 1:1 인간 손 모사, 구체적 미공개 | 공개 자료 |
| 무게 | 500 g | 공개 자료 |
| 손가락 수 | 5 | 공개 자료 |
| 자유도 | 19 관절 / 1 능동 협력(단일 모터) | 공개 자료 |
| 손끝 파지력 | 62 N | 공개 자료 |
| 정격 하중 | 1.7 kg(파지 상태) | 공개 자료 |
| 펴기에서 주먹 쥐기까지 | 1.1 s | 공개 자료 |
| 통신 인터페이스 | USB / RS485 | 공개 자료 |
| 호환성 | ROS, UR+ 인증 | 공개 자료 |
| 가격 | 미공개 | - |

**기술적 특징**: 소프트 소재와 텐던 구동 협력, 자체 리셋 관절, 단일 모터 플러그 앤 플레이, 높은 하중 대 자체 중량비, 안전한 인간-로봇 상호작용.

**적용 시나리오**: 연구용 파지, 교육 시연, 협동 로봇 엔드 이펙터, 의료 재활, 인간-로봇 상호작용.


### qb SoftHand2 Research / qb SoftHand2 Research

> qb SoftHand2 Research: [공식 자료](https://qbrobotics.com)를 방문하여 확인하세요.

| 사양 항목 | 수치 | 비고/출처 |
|--------|------|-----------|
| 크기 | 미공개 | - |
| 무게 | 0.94 kg | 공개 자료 |
| 자유도 | 19 관절 / 2 모터 / 2 협력 | 공개 자료 |
| 정격 하중 | 2 kg(집기) / 3 kg(쥐기) | 공개 자료 |
| 펴기에서 주먹 쥐기까지 | 약 1 s | 공개 자료 |
| 통신 인터페이스 | USB / RS485 | 공개 자료 |
| 호환성 | ROS / ROS2 | 공개 자료 |
| 가격 | 미공개 | - |

**기술적 특징**: 두 번째 협력 동작으로 손 안에서의 조작 능력 향상, 손목 회전 없이 물체 자세 조정 가능.

**적용 시나리오**: 고급 파지 연구, 산업 테스트, 인간-로봇 협력, 복잡한 물체 조작.


## 공급망 위치

- **상류 핵심 부품/소재**: 엘라스토머 소재, 텐던/소프트 구동, 모터, 센서, 3D 프린팅 구조 부품

- **주요 경쟁사/대상**: Shadow Robot, Pisa/IIT SoftHand, Robotiq, Agile Robots
