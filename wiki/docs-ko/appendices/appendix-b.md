# 부록 B 주요 데이터셋 목록

> 본 부록은 `scripts/build_appendices.py` 스크립트가 지식 그래프 데이터로부터 자동 생성하며, KG 업데이트에 따라 갱신됩니다.

지식 그래프에 수록된 11개 주요 데이터셋 엔터티는 원격 조작 시연, 1인칭 시점 비디오, 모션 캡처 및 대규모 교차 임바디먼트 데이터를 포괄합니다.


| 이름 | 영문명 | 개요 |
|---|---|---|
| [AMASS 모션 데이터셋](/entry/ent_dataset_amass_motion_dataset_2019/) | AMASS Motion Dataset | 인체 형상과 자세를 포함한 대규모 정제 모션 캡처 데이터셋으로, 휴머노이드 로봇 시뮬레이션 사전 학습에 널리 사용됩니다. |
| [ARIO 전 로봇 통합 데이터셋](/entry/ent_dataset_ario_dataset_2024/) | ARIO Dataset | 통합 데이터 표준으로, 약 300만 개의 클립을 포함하며 실제, 시뮬레이션 및 변환된 로봇 데이터 소스를 융합합니다. |
| [BridgeData V2](/entry/ent_dataset_bridgedata_v2_2023/) | BridgeData V2 | 모방 학습에 널리 사용되는 언어 명령이 포함된 데스크톱 조작 데이터셋입니다. |
| [DROID 로봇 조작 데이터셋](/entry/ent_dataset_droid/) | DROID | 여러 연구소 및 환경에서 수집된 분산형 다양성 있는 실제 로봇 조작 데이터셋으로, 시각 및 환경 강건성 향상을 목표로 합니다. |
| [Ego4D 1인칭 시점 비디오 데이터셋](/entry/ent_dataset_ego4d_egocentric_video_dataset_2022/) | Ego4D Egocentric Video Dataset | 3000시간 이상의 1인칭 시점 비디오로, 1인칭 스킬 및 상호작용 이해에 사용됩니다. |
| [Epic-Kitchens 데이터셋](/entry/ent_dataset_epic_kitchens_dataset_2018/) | Epic-Kitchens Dataset | 1인칭 시점 주방 상호작용 비디오 데이터셋으로, 조작 및 활동 이해에 널리 사용됩니다. |
| [HuDOR](/entry/ent_hudor/) | HuDOR | 사람 손 동작 데이터를 수집하여 손재주 있는 로봇 손의 모방 학습 정책을 훈련하기 위한 인간-로봇 시연 데이터셋 또는 원격 조작 프레임워크입니다. |
| [HumanPlus 섀도잉 데이터셋](/entry/ent_dataset_humanplus_shadowing/) | HumanPlus Shadowing Dataset | 휴머노이드 로봇이 인체 동작을 추종하여 수집한 데이터셋으로, 전신 모방 및 원격 조작 정책 훈련에 사용됩니다. |
| [OmniAction 데이터셋](/entry/ent_dataset_omniaction/) | OmniAction Dataset | 능동적 로봇 조작을 위한 대규모 다중 모달 데이터셋으로, 14. |
| [Open X-Embodiment 데이터셋](/entry/ent_dataset_open_x_embodiment/) | Open X-Embodiment Dataset | 개요: 다양한 실제 로봇 플랫폼 및 기관의 시연 데이터를 통합한 대규모 집계 로봇 학습 데이터셋으로, 교차 임바디먼트 VLA 사전 학습에 널리 사용됩니다. |
| [RH20T 조작 데이터셋](/entry/ent_dataset_rh20t_manipulation_dataset_2023/) | RH20T Manipulation Dataset | 약 11만 개의 접촉이 풍부한 조작 시퀀스로, 시각, 힘, 오디오 및 인간 시연 쌍을 포함합니다. |
