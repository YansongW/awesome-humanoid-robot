# 北方稀土 钕铁硼速凝薄带合金 / CNRE NdFeB Strip Casting Alloy

## 概述

北方稀土钕铁硼速凝薄带合金是中国北方稀土（集团）高科技股份有限公司（CNRE）旗下磁材公司生产的烧结钕铁硼前驱体材料。该产品通过速凝铸带（Strip Casting，SC）工艺将熔融 NdFeB 合金快速凝固成薄带，是制造高性能烧结钕铁硼磁体的关键原材料。北方稀土磁材公司北京三吉利公司已开发出具有自主知识产权的高品质速凝薄带制备技术。

## 物理化学基础

### 晶体组织与形成机制
速凝薄带工艺将合金熔体浇注到高速旋转的冷却辊上，实现快速凝固（冷却速率约 10⁵–10⁶ K/s）。该工艺可抑制 α-Fe 偏析与富 Nd 相聚集，使主相 Nd₂Fe₁₄B 以片状柱状晶形式均匀分布，富 Nd 相沿晶界薄层分布。典型微观组织参数：

| 参数 | 典型值 |
|------|--------|
| 薄带厚度 | 0.1–0.6 mm（主流 0.2–0.4 mm） |
| 柱状晶平均宽度 | 1–5 μm |
| 柱状晶区比例 | ≥80% |
| 主相片层厚度 | 3–7 μm |
| 非晶/等轴晶缺陷 | 应尽量避免 |

### 成分设计
典型钕铁硼速凝薄带合金名义成分（质量分数）示例：

| 元素 | 典型范围（质量%） |
|------|------------------|
| Nd | 25–30% |
| Pr | <1–5% |
| Dy | 0.5–2% |
| Fe | 余量 |
| B | 0.9–1.1% |
| Al | 0.3–0.6% |
| Co/Cu/Ga/Nb/Zr | 微量添加 |

Dy 等重稀土添加可提高烧结磁体的矫顽力与温度稳定性。

### 工艺动力学
熔体在冷却辊表面的凝固过程可用一维非稳态热传导近似：

$$
\frac{\partial T}{\partial t} = \alpha \frac{\partial^2 T}{\partial x^2}
$$

其中 $\alpha = k/(\rho c_p)$ 为热扩散系数，$k$ 为热导率，$\rho$ 为密度，$c_p$ 为比热容。辊速增加会提高冷却速率，使薄带厚度减薄、柱状晶细化，但过高冷却速率会在急冷面产生细小等轴晶，降低烧结磁体性能。

### 国家标准要求
根据 GB/T 29655（钕铁硼速凝薄片合金）修订方向：

- 增加平均柱状晶宽度指标：1–5 μm；
- 薄带厚度范围调整为 0.1–0.6 mm；
- 柱状晶区比例由不少于 60% 提升至不少于 80%；
- 应不含 α-Fe 相，富 Nd 相沿晶界均匀分布。

## 在机器人系统中的作用

- **上游原材料**：为烧结钕铁硼磁钢制造商提供高性能合金薄带；
- **磁体性能基础**：速凝薄带的微观组织直接决定后续氢破碎、气流磨磁粉的粒度分布与取向度，进而影响烧结磁体的剩磁、矫顽力与最大磁能积；
- **供应链上游**：位于“稀土矿—稀土金属—速凝薄带—烧结磁钢—电机—机器人”链条的速凝薄带层；
- **系统影响**：高品质速凝薄带可提升磁体一致性、降低重稀土用量、降低电机成本，从而推动机器人关节电机的大规模应用。

## 供应链关系

北方稀土钕铁硼速凝薄带合金位于稀土永磁材料供应链上游：

- **上游**：稀土矿山（白云鄂博矿）、稀土冶炼分离（北方稀土）；
- **中游**：速凝薄带合金生产（北方稀土磁材公司北京三吉利）、磁粉/磁钢制造（金力永磁、中科三环、宁波韵升等）；
- **下游**：永磁电机、机器人整机、新能源汽车、风力发电、节能家电；
- **合作伙伴**：下游磁体制造厂家、科研院所。
